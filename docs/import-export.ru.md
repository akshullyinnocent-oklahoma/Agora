# Переносимость данных

Agora хранит все ваши данные на устройстве и предоставляет полные возможности импорта/экспорта. Вы владеете своими данными — перемещайте их внутрь, наружу, создавайте резервные копии.

## Экспорт

Экспортируйте ваши данные в один файл `.agora` — переносимый архив, содержащий всё, что хранит Agora.

### Что экспортируется

Вы выбираете, что включить:

| Категория | Содержимое |
|----------|----------|
| **Диалоги и сообщения** | Вся история чата, деревья сообщений, ветви |
| **Воспоминания** | Активная память и все сохранённые файлы памяти |
| **Системные промпты** | Все пользовательские шаблоны системных промптов |
| **Настройки** | Конфигурация приложения и предпочтения |
| **API-ключи** | Все настроенные API-ключи |

!!! danger "Предупреждение об API-ключах"
    API-ключи экспортируются в **открытом виде**. Любой, у кого есть файл `.agora`, может прочитать ваши ключи. Включайте экспорт API-ключей только если вы доверяете получателю и обращаетесь с файлом безопасно.

### Как экспортировать

1. Перейдите в **Настройки → Управление данными**
2. Нажмите **Экспорт данных**
3. Выберите, какие категории включить
4. Нажмите **Экспорт**
5. Выберите, куда сохранить файл `.agora`

## Импорт

Восстановите данные из предыдущего экспорта `.agora`.

### Стратегии импорта

При импорте вы выбираете, как Agora обрабатывает данные, уже существующие на вашем устройстве:

| Стратегия | Поведение |
|----------|----------|
| **Слияние** | Добавить новые элементы, сохранить существующие. Если элемент с тем же ID существует, версия из импорта перезаписывает его. |
| **Замена** | Очистить все существующие данные в выбранных категориях, затем импортировать. Чистый старт. |
| **Пропуск** | Импортировать только элементы без конфликтов. Существующие элементы не затрагиваются. |

!!! tip
    Используйте **Слияние** в большинстве случаев — оно безопасно добавляет новые данные, сохраняя то, что уже есть на устройстве.

### Как импортировать

1. Перейдите в **Настройки → Управление данными**
2. Нажмите **Импорт данных**
3. Выберите файл `.agora`
4. Просмотрите предпросмотр импорта — увидите, что в файле (дата экспорта, версия, количество содержимого)
5. Выберите стратегию импорта
6. Нажмите **Импорт**

!!! danger "Предупреждение об API-ключах"
    Если файл экспорта содержит API-ключи, Agora предупреждает вас перед импортом. Ключи импортируются в открытом виде. Продолжайте, только если вы доверяете источнику файла.

## Сторонний импорт

Импортируйте диалоги из других платформ AI-чатов.

Claude и ChatGPT экспортируют ваши данные в виде **архива `.zip`**. Agora импортирует этот `.zip` напрямую — распаковывать его предварительно не нужно, и Agora **не** принимает отдельные файлы `.json`.

### Импорт из Claude

**1. Экспорт из Claude.** Перейдите в [Claude](https://claude.ai/) → **Настройки → Управление данными → Экспорт данных**. Claude подготавливает архив быстро — обычно **менее чем за час** — и отправляет вам ссылку для скачивания по электронной почте.

!!! warning "Скачайте своевременно"
    Ссылка для скачивания Claude **быстро истекает**. Заберите `.zip`, как только придёт письмо — если ждать слишком долго, ссылка станет недействительной, и придётся запрашивать новый экспорт.

**2. Импорт в Agora.**

1. Перейдите в **Настройки → Управление данными → Сторонние → Импорт из Claude**
2. Выберите экспортированный файл `.zip`
3. Просмотрите предпросмотр — увидите количество диалогов и сообщений
4. Выберите стратегию **Слияние** или **Замена**
5. Нажмите **Импорт**

!!! note
    Agora читает данные диалогов напрямую из `.zip` экспорта Claude. Вложения обнаруживаются и показываются в предпросмотре, но импортируется только текст сообщений — сами файлы вложений не импортируются.

### Импорт из ChatGPT

**1. Экспорт из ChatGPT.** Перейдите в [ChatGPT](https://chatgpt.com/) → **Настройки → Управление данными → Экспорт данных**. ChatGPT обрабатывает запрос и отправляет вам ссылку для скачивания по электронной почте, когда всё готово.

!!! info "Будьте терпеливы"
    Экспорт ChatGPT обычно занимает **1–2 дня**. Это нормально — дождитесь письма, а не запрашивайте повторно.

**2. Импорт в Agora.**

1. Перейдите в **Настройки → Управление данными → Сторонние → Импорт из ChatGPT**
2. Выберите скачанный файл `.zip`
3. Просмотрите предпросмотр
4. Выберите стратегию **Слияние** или **Замена**
5. Нажмите **Импорт**

!!! note
    Импортируются как сообщения пользователя, так и ассистента. Роли сообщений сохраняются.

## Формат файла

Файл `.agora` — это архив на основе JSON. Если вы технически подкованы, вы можете проверить или обработать его стандартными инструментами. Формат разработан для прямой и обратной совместимости.

## Авто-резервное копирование

Agora может автоматически создавать резервные копии ваших данных по расписанию. Вам не нужно помнить об экспорте — Agora сделает это за вас.

### Как это работает

- Авто-резервное копирование периодически выполняется в фоновом режиме с использованием Android WorkManager
- Когда подходит время резервной копии, Agora экспортирует выбранные вами категории в настроенную директорию
- Уведомление появляется только при сбое резервной копии — успешные резервные копии выполняются бесшумно
- Старые резервные копии автоматически удаляются в соответствии с вашими настройками хранения

### Настройка

1. Перейдите в **Настройки → Управление данными → Авто-резервное копирование**
2. Переключите **Авто-резервное копирование** вкл/выкл
3. Установите **Резервное копирование каждые** — выберите 1 день, 3 дня, 5 дней, 1 неделю или 1 месяц
4. Выберите **Содержимое экспорта** — выберите категории для включения. API-ключи **могут** быть включены (предупреждение показывается при установке этого флажка) — включайте только если место резервного копирования является приватным и безопасным. API-ключи **не** включены по умолчанию.
5. Установите **Место резервной копии** — нажмите для выбора папки (по умолчанию `Download/Agora/Backup`)
6. Переключите **Авто-удаление старых резервных копий** вкл/выкл и установите период **Удалять старше**

!!! info "Ограничение авто-удаления"
    Период удаления должен быть больше периода резервного копирования. Например, если вы создаёте резервные копии каждую неделю, они могут быть авто-удалены через 1 месяц или 1 год — никогда раньше. Это предотвращает удаление единственной резервной копии до создания новой.

!!! note
    Авто-резервное копирование использует Android WorkManager для обеспечения надёжности, даже если приложение закрыто или устройство перезагружено. Резервные копии могут немного задерживаться во время режима Doze для экономии заряда батареи.

## Лучшие практики

- **Экспортируйте регулярно** в качестве резервной копии — храните файл в надёжном месте
- **Включите Авто-резервное копирование** для автоматической защиты по расписанию
- **Не включайте API-ключи** в рутинные экспорты — включайте экспорт ключей только для полной миграции устройства
- **Используйте Слияние для инкрементальных импортов** — Замена разрушительна
- **Предпросматривайте перед импортом** — проверьте дату экспорта и количество содержимого, чтобы убедиться, что это правильный файл
