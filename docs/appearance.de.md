# Erscheinungsbild

Passen Sie Agoras Aussehen an — Theme-Modus, Farbschema und Stil.

## Theme-Modus

Wählen Sie zwischen hellem und dunklem Erscheinungsbild.

| Modus | Beschreibung |
|------|-------------|
| **System** | Folgt dem System-Theme Ihres Geräts (Standard) |
| **Hell** | Immer helles Theme verwenden |
| **Dunkel** | Immer dunkles Theme verwenden |

Änderungen werden sofort wirksam.

---

## Farbschema

Wählen Sie aus einer kuratierten Auswahl von Farbpaletten, die auf Material 3 Design-Prinzipien basieren.

| Schema | Charakter |
|--------|-----------|
| **Midnight** | Tiefe Blautöne, ruhig und professionell (Standard) |
| **Nordic** | Kühles Türkis und Schiefer, klar und minimal |
| **Forest** | Warme Grün- und Erdtöne, natürliches Gefühl |
| **Sunset** | Orange und warme Töne, energetisch |
| **Rose** | Rosa und Rosenakzente, weich und warm |
| **Lavender** | Lila und Violetttöne, elegant |
| **Slate** | Neutrale Grau- und Blautöne, dezent |
| **Ocean** | Tiefblau und Cyan, erfrischend |

Jedes Schema hat sowohl helle als auch dunkle Varianten.

---

## Schema-Stil

Der Schema-Stil bestimmt, *wie* Farben aus der gewählten Palette auf UI-Elemente angewendet werden.

| Stil | Effekt |
|-------|--------|
| **Tonal Spot** | Primärfarbakzente auf Schlüsselflächen (Standard) |
| **Expressive** | Kräftiger Einsatz von Akzentfarben durchgehend |
| **Vibrant** | Hochgesättigte Farben, lebhaftes Erscheinungsbild |
| **Neutral** | Minimale Farbe, überwiegend Oberflächentöne |

---

## Dynamic Color (Android 12+)

Wenn auf Geräten mit Android 12 oder höher aktiviert, extrahiert Agora eine Farbpalette aus Ihrem System-Hintergrundbild und wendet sie automatisch an.

Dies überschreibt die manuelle **Farbschema**-Auswahl. Deaktivieren Sie es, um ein benutzerdefiniertes Schema zu verwenden.

!!! info "Verfügbarkeit"
    Dynamic Color erfordert Android 12 (API 31) oder höher. Auf älteren Geräten ist der Schalter ausgeblendet.
