# التضمين / RAG

نماذج التضمين تحول النص إلى متجهات عددية تلتقط المعنى. يستخدم Agora هذه المتجهات للبحث الدلالي (RAG) عبر سجل محادثاتك — إيجاد الرسائل بما تعنيه، وليس فقط بالكلمات التي تحتويها.

## كيف يعمل

1. كل رسالة ترسل إلى نموذج تضمين
2. النموذج يرجع متجهًا (قائمة أرقام) يمثل معنى الرسالة
3. عندما تبحث، استعلامك يُضمن أيضًا
4. يحسب Agora **تشابه جيب التمام** بين متجه الاستعلام وكل متجهات الرسائل
5. الرسائل ذات التشابه فوق عتبتك ترجع كمطابقات

## المزودون المدعومون

| المزود | URL الأساسي | يتطلب مفتاح API | ملاحظات |
|----------|----------|------------------|-------|
| **OpenAI** | `https://api.openai.com/v1` | نعم | `text-embedding-3-small`، `text-embedding-3-large` |
| **Mistral** | `https://api.mistral.ai/v1` | نعم | `mistral-embed` |
| **Voyage AI** | `https://api.voyageai.com/v1` | نعم | `voyage-3`، `voyage-3-lite` |
| **SiliconFlow** | `https://api.siliconflow.cn/v1` | نعم | `BAAI/bge-large-zh-v1.5` (محسن للصينية) |
| **Ollama** | `http://localhost:11434/v1` | لا | `qwen3-embedding`، `nomic-embed-text`، إلخ. |
| **مخصص** | أي | اختياري | أي نقطة نهاية تضمين متوافقة مع OpenAI |
| **محلي** | لا ينطبق | لا | نماذج تضمين GGUF عبر llama.cpp |

## إضافة نموذج تضمين

### عن بُعد (API)

1. اذهب إلى **الإعدادات → البحث في المحادثات**
2. اضغط **إضافة نموذج عن بُعد**
3. إعداد:

| الحقل | الوصف |
|-------|-------------|
| **المزود** | اختر من القائمة المنسدلة (OpenAI، Mistral، Voyage، SiliconFlow، Ollama، مخصص) |
| **اسم النموذج** | معرف النموذج الدقيق (مثل `text-embedding-3-small`) |
| **URL الأساسي** | مملوء تلقائيًا للمزودين المعروفين؛ قابل للتعديل للبروكسيات |
| **مفتاح API** | اتركه فارغًا للحل التلقائي من مفتاح مزود الدردشة، أو أدخل مفتاحًا مخصصًا |
| **حجم الدفعة** | الرسائل المراد تضمينها لكل طلب API (1–100) |

4. اضغط **إضافة** — يُجرى اختبار اتصال قبل الحفظ

!!! tip
    حقل مفتاح API اختياري إذا كنت قد أعددت مسبقًا نفس المزود للدردشة. اتركه فارغًا وسيحل Agora مفتاح API للدردشة تلقائيًا.

### محلي (GGUF)

1. اذهب إلى **الإعدادات → البحث في المحادثات**
2. اضغط **إضافة نموذج محلي**
3. استورد ملف نموذج تضمين `.gguf` (مثل `bge-small-en-v1.5-q4_k.gguf`)
4. أعطه اسمًا
5. اضغط **إضافة**

نماذج التضمين عادة أصغر بكثير من نماذج الدردشة — بضع مئات من MB على الأكثر.

### Ollama

1. ثبّت Ollama على جهاز
2. اسحب نموذج تضمين: `ollama pull qwen3-embedding:8b`
3. في Agora، أضف نموذجًا عن بُعد:
    - المزود: **Ollama**
    - URL الأساسي: `http://<host>:11434/v1`
    - اسم النموذج: `qwen3-embedding:8b` (ضمن الوسم `:tag`)
    - مفتاح API: اتركه فارغًا
4. اضغط **إضافة**

!!! note
    وسوم لاحقة Ollama مثل `:8b`، `:latest` هي جزء من اسم النموذج. استخدم الاسم الدقيق من `ollama list`.

## التخزين المؤقت

بعد إضافة نموذج، تحتاج إلى تخزين رسائلك مؤقتًا (توليد التضمينات):

1. اضغط **تخزين مؤقت** على نموذج التضمين
2. يعالج Agora كل الرسائل غير المخزنة في دفعات
3. مؤشر تقدم دائري يظهر التقدم الحالي
4. الاكتمال: "كل N رسالة مخزنة"

### التخزين المؤقت التلقائي

فعّل **التخزين المؤقت التلقائي** لتضمين الرسائل الجديدة تلقائيًا فور وصولها. هذا يحافظ على تحديث فهرس البحث دائمًا.

### إعادة التخزين المؤقت

اضغط **إعادة التخزين المؤقت** لحذف كل التضمينات الموجودة وإعادة البناء من الصفر. استخدمه عندما:

- تنتقل إلى نموذج تضمين مختلف
- جودة التضمين تبدو منخفضة
- المخزن المؤقت غير متسق

!!! warning
    إعادة التخزين المؤقت لا يمكن التراجع عنها وقد تستغرق وقتًا طويلاً لتواريخ الرسائل الكبيرة.

## حجم الدفعة

إعداد **حجم الدفعة** (1–100) يتحكم في كم رسالة ترسل لكل طلب API أثناء التخزين المؤقت:

- **أعلى**: تخزين مؤقت أسرع، لكن حمولات API أكبر
- **أقل**: طلبات أصغر، أبطأ لكن أكثر موثوقية على الاتصالات البطيئة

ابدأ بالافتراضي وعدّل إذا واجهت انتهاء مهلة (اخفضه) أو أردت تخزينًا أسرع (ارفعه).

## اختبار إعدادك

عند إضافة نموذج عن بُعد، يجري Agora اختبار اتصال تلقائي. إذا فشل:

1. تحقق من اسم النموذج — ضمن الوسوم لـ Ollama (`:8b`، `:latest`)
2. تحقق أن URL الأساسي قابل للوصول من جهازك
3. تأكد أن مفتاح API صالح (إذا كان مطلوبًا)
4. جرب اسم نموذج معروف لذلك المزود

الأخطاء الشائعة:
- **"اسم نموذج خاطئ"** — تحقق من التهجئة الدقيقة، بما في ذلك الوسوم
- **"URL أساسي خاطئ"** — تأكد أن نقطة النهاية تدعم `/v1/embeddings`
- **"مفتاح API مفقود"** — بعض المزودين يتطلبون توثيقًا
- **"خطأ شبكة"** — تحقق من الاتصال
