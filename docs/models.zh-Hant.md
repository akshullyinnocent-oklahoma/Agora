# 模型

管理哪些 AI 模型可用，並設定對話的預設模型。

## 模型清單

**模型** 頁面顯示 Agora 已知的所有模型，按提供者分類：

- **預設模型** — 新對話使用的模型。點選可變更。
- **可用模型** — 展開每個提供者查看其模型。啟用您想使用的模型。

### 啟用 / 停用模型

勾選或取消勾選模型旁的核取方塊來切換其可用性。已停用的模型不會出現在對話的模型選擇器中。

### 重新命名模型

點選模型旁的編輯圖示（筆）來給予自訂別名。此別名會在整個應用程式中顯示，取代技術性的模型 ID。

### 同步模型

點選 **同步模型** 從所有已設定的 API 提供者取得最新可用模型。這需要網路連線和有效的 API 金鑰。

!!! tip "本機模型"
    本機模型顯示在 **本機** 提供者區段下。它們在 **設定 → 提供者 → 本機** 中單獨管理。

---

## 預設模型

**預設模型** 用於所有新對話。變更方式：

1. 點選模型頁面頂部的預設模型列
2. 從清單中選擇模型（僅顯示已啟用的模型）
3. 變更立即生效

您可以從聊天畫面的模型選擇器為個別對話覆寫模型。

---

## 模型別名

模型別名讓您可以為具有冗長技術 ID 的模型取一個友善的名稱。例如，您可以將 `openai/gpt-4o-mini` 重新命名為「GPT-4o Mini」。

別名會顯示在所有地方：模型選擇器、對話標題和設定頁面。

要移除別名，清空文字欄位並儲存。

---

## 疑難排解

### 模型未出現

- 點選 **同步模型** 重新整理清單
- 確認您在 **設定 → 提供者** 中為該提供者設定了有效的 API 金鑰
- 檢查您的網路連線
- 某些提供者可能暫時無法使用

### 本機模型未顯示

- 在 **設定 → 提供者 → 本機** 中匯入 GGUF 模型檔案
- 模型必須是有效的 GGUF 格式
