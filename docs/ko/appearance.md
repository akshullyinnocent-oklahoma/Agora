# 외관

Agora의 룩앤필 — 테마 모드, 색상 구성, 스타일을 사용자 지정하세요.

## 테마 모드

라이트와 다크 외관 중에서 선택하세요.

| 모드 | 설명 |
|------|-------------|
| **시스템** | 기기의 시스템 테마를 따릅니다 (기본값) |
| **라이트** | 항상 라이트 테마 사용 |
| **다크** | 항상 다크 테마 사용 |

변경 사항은 즉시 적용됩니다.

---

## 색상 구성

Material 3 디자인 원칙을 기반으로 한 엄선된 색상 팔레트 세트에서 선택하세요.

| 구성 | 특징 |
|--------|-----------|
| **Midnight** | 깊은 블루 톤, 차분하고 전문적인 느낌 (기본값) |
| **Nordic** | 시원한 틸과 슬레이트, 깨끗하고 미니멀 |
| **Forest** | 따뜻한 그린과 어스 톤, 자연스러운 느낌 |
| **Sunset** | 오렌지와 웜 톤, 활기찬 느낌 |
| **Rose** | 핑크와 로즈 악센트, 부드럽고 따뜻함 |
| **Lavender** | 퍼플과 바이올렛 색조, 우아함 |
| **Slate** | 중성적인 그레이와 블루, 절제된 느낌 |
| **Ocean** | 깊은 블루와 시안, 상쾌함 |

각 구성에는 라이트 및 다크 변형이 모두 있습니다.

---

## 구성 스타일

구성 스타일은 선택한 팔레트의 색상이 UI 요소에 *어떻게* 적용되는지를 결정합니다.

| 스타일 | 효과 |
|-------|--------|
| **Tonal Spot** | 주요 표면에 기본 색상 악센트 (기본값) |
| **Expressive** | 전체에 걸친 대담한 악센트 색상 사용 |
| **Vibrant** | 고채도 색상, 활기찬 외관 |
| **Neutral** | 최소한의 색상, 주로 표면 톤 |

## 블러 효과

**블러 효과 활성화**를 토글하여 UI 전체의 배경 블러와 그라데이션 블러를 제어합니다. 비활성화 시, 블러 효과는 단색으로 대체되어 저사양 기기에서 성능이 향상될 수 있습니다.

!!! tip "성능"
    구형 기기에서 스크롤이나 애니메이션 중 UI 지연이 느껴지면 블러 효과를 비활성화하세요.

---

## 동적 색상 (Android 12+)

Android 12 이상 기기에서 활성화 시, Agora는 시스템 배경 화면에서 색상 팔레트를 추출하여 자동으로 적용합니다.

이는 수동 **색상 구성** 선택을 덮어씁니다. 사용자 지정 구성을 사용하려면 비활성화하세요.

!!! info "이용 가능 여부"
    동적 색상은 Android 12 (API 31) 이상이 필요합니다. 구형 기기에서는 토글이 숨겨집니다.
