## 업데이트

### 업데이트 확인

**업데이트 확인**을 탭하여 GitHub에서 최신 릴리스를 조회합니다. Agora는 설치된 버전과 최신 릴리스를 비교하여 결과를 표시합니다:

- **업데이트 가능** — GitHub에 새 버전이 게시됨
- **최신 상태** — 최신 릴리스 실행 중

### 자동 업데이트 확인

활성화 시, Agora는 백그라운드에서 주기적으로 새 릴리스를 확인합니다. 자동 확인을 비활성화하려면 이 토글을 끄세요.

---

## 문서

### 문서 링크 표시

설정 하위 페이지에 나타나는 **플로팅 도움말 버튼**(FAB)을 제어합니다. 활성화 시, :material-help-circle: 버튼이 관련 사용자 매뉴얼 페이지로 연결됩니다.

문서 FAB 없이 더 깔끔한 인터페이스를 원하시면 이 토글을 끄세요.

## 크래시 로그

Agora에서 오류가 발생하면 **설정 → 정보 → 크래시 로그**에서 크래시 로그를 볼 수 있습니다. 로그는 모노스페이스 대화 상자에 표시되며 **복사** 버튼으로 버그 보고 시 진단 정보를 공유할 수 있습니다.
