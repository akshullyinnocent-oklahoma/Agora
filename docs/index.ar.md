# دليل مستخدم Agora

مرحبًا بك في دليل مستخدم Agora. Agora هو تطبيق BYOK (أحضر مفتاحك الخاص) للدردشة مع نماذج اللغة الكبيرة على Android مع دعم متعدد المزودين، محادثات متفرعة غير خطية، استدعاء أدوات وكيلية، والتحكم بالأجهزة عن بُعد.

## حول Agora

Agora هو تطبيق BYOK لنظام Android لمستخدمي الذكاء الاصطناعي المتقدمين:

- **بدون وسطاء**: اتصالات API مباشرة، لا تتبع، لا مراقبة
- **تخزين على الجهاز**: كل شيء محفوظ محليًا في قاعدة بيانات Room
- **محادثات غير خطية**: عدّل أي رسالة سابقة واستكشف مسارات بديلة
- **وكيلي افتراضيًا**: استدعاء أدوات متعدد الجولات مع البحث على الويب، تنفيذ الأوامر، عمليات الملفات، والذاكرة
- **تحكم عن بُعد**: إدارة الخوادم عبر بروتوكول Conch المشفر
- **مفتوح المصدر**: مرخص تحت MIT، [المصدر على GitHub](https://github.com/newo-ether/Agora)
