# البحث على الويب

دع النموذج يبحث في الإنترنت ويجلب صفحات الويب في الوقت الفعلي. عند التفعيل، يمكن للنموذج البحث عن معلومات حالية، التحقق من الحقائق، استرجاع التوثيق، أو البحث في مواضيع — كل ذلك ذاتيًا عبر استدعاء الأدوات.

## المزودون المدعومون

| المزود | الوصف | فئة مجانية | الإعداد |
|----------|-------------|-----------|-------|
| **DuckDuckGo Lite** | مجهول، لا حاجة لمفتاح API | نعم (غير محدود، بأفضل جهد) | بلا إعداد — يعمل مباشرة |
| **Brave** | API بحث مركّز على الخصوصية | نعم (2,000 استعلام/شهر) | [api.search.brave.com](https://api.search.brave.com/) |
| **Serper** | API بحث Google سريع | نعم (2,500 استعلام/شهر) | [serper.dev](https://serper.dev) |
| **Tavily** | بحث محسن للذكاء الاصطناعي، مبني لوكلاء LLM | نعم (1,000 استعلام/شهر) | [tavily.com](https://tavily.com) |
| **SearXNG** | محرك بحث ميتا مستضاف ذاتيًا | مستضاف ذاتيًا (غير محدود) | نسختك الخاصة |

## الإعداد

### DuckDuckGo Lite

DuckDuckGo Lite هو مزود البحث **الافتراضي** — لا حاجة لمفتاح API، يعمل فورًا.

1. في Agora، اذهب إلى **الإعدادات → البحث على الويب**
2. اختر **DuckDuckGo Lite** كمزود البحث
3. لا حاجة لمفتاح أو URL — ابدأ البحث فورًا

!!! note "خدمة بأفضل جهد"
    DuckDuckGo Lite يستخدم كشط HTML من `lite.duckduckgo.com`. قد تغير DDG تخطيطها، أو تفرض قيودًا على المعدل، أو تحظر الطلبات الآلية. يُقدم كخيار صريح بأفضل جهد وبدون مفتاح. إذا كنت بحاجة إلى موثوقية، فقم بإعداد أحد المزودين المعتمدين على API أدناه.

### Brave

1. احصل على مفتاح API من [Brave Search API](https://api.search.brave.com/)
2. في Agora، اذهب إلى **الإعدادات → البحث على الويب**
3. اختر **Brave** كمزود البحث
4. الصق مفتاح API الخاص بك

### Serper

1. احصل على مفتاح API من [serper.dev](https://serper.dev)
2. في Agora، اذهب إلى **الإعدادات → البحث على الويب**
3. اختر **Serper**
4. الصق مفتاح API الخاص بك

### Tavily

1. احصل على مفتاح API من [tavily.com](https://tavily.com)
2. في Agora، اذهب إلى **الإعدادات → البحث على الويب**
3. اختر **Tavily**
4. الصق مفتاح API الخاص بك

### SearXNG

1. أنشئ نسخة SearXNG (مستضافة ذاتيًا) أو استخدم نسخة عامة
2. في Agora، اذهب إلى **الإعدادات → البحث على الويب**
3. اختر **SearXNG**
4. أدخل **URL الأساسي** لنسختك (مثل `https://searx.be`)
5. مفتاح API اختياري (مطلوب فقط إذا كانت نسختك تتطلب توثيقًا)

!!! warning "النسخ العامة"
    نسخ SearXNG العامة غالبًا محدودة المعدل أو غير موثوقة. الاستضافة الذاتية موصى بها للاستخدام المتسق.

## الإعداد

### الحد الأقصى للنتائج

حدد كم نتيجة بحث تجلب لكل استعلام: **1–10**. الافتراضي يعتمد على الجهاز. نتائج أكثر تعطي النموذج سياقًا أكثر لكنها تكلف رموزًا أكثر.

### التفعيل/التعطيل

بدّل **تفعيل البحث على الويب** في صفحة إعدادات البحث على الويب. عند التعطيل، لا يمكن للنموذج استدعاء أداة البحث على الويب.

---

## كيف يستخدم النموذج البحث

عندما تسأل سؤالاً يحتاج معلومات حالية أو خارجية، يستدعي النموذج البحث على الويب تلقائيًا:

1. **البحث**: النموذج يستدعي API البحث باستعلام يصيغه
2. **الجلب**: النموذج يمكنه اختياريًا جلب محتوى الصفحة الكامل من URLs النتائج
3. **التركيب**: النموذج يقرأ النتائج ويدمجها في رده

سترى كل بحث وجلب كبطاقات أداة مضمنة في المحادثة.

### مثال

```text
أنت: "ما هو أحدث إصدار من Python؟"
النموذج: [يبحث "أحدث إصدار Python 2026"]
       [يقرأ النتيجة]
       "تم إصدار Python 3.14.0 في أكتوبر 2025..."
```

---

## جلب الويب

أبعد من البحث، يمكن للنموذج جلب وقراءة صفحات ويب محددة. عندما يواجه النموذج URL في نتائج البحث، يمكنه استدعاء `web_fetch` لاسترجاع محتوى الصفحة الكامل:

- المحتوى المجلوب يحول إلى markdown
- النموذج يعالجه ويستخرج المعلومات ذات الصلة
- نتائج الجلب تظهر كبطاقات أداة

---

## اعتبارات الخصوصية

عند استخدام البحث على الويب:

- استعلاماتك تذهب إلى مزود البحث (Brave، Serper، إلخ.)، وليس إلى Agora
- Agora لا يسجل أو يخزن استعلامات البحث الخاصة بك (باستثناء المحادثة نفسها)
- الاستضافة الذاتية لـ SearXNG تمنحك أكبر خصوصية — الاستعلامات تبقى على بنيتك التحتية
