# 言語

Agora のインターフェースの表示言語を選択します。

## 利用可能な言語

| 言語 | コード | 説明 |
|----------|------|-------------|
| **システムデフォルト** | `system` | デバイスのシステム言語設定に従います |
| **English** | `en` | すべての UI テキストが英語 |
| **简体中文** | `zh` | すべての UI テキストが簡体字中国語 |

## 変更方法

1. **設定 → 言語** に移動します
2. 希望の言語を選択します
3. スナックバーが表示されます：**再起動** をタップして変更をすぐに適用します

!!! note "再起動が必要です"
    言語の変更にはアプリの完全な再起動が必要です。新しい言語を選択した後、Agora は再起動を促します — スナックバーの再起動アクションをタップして新しい言語でアプリをリロードします。
