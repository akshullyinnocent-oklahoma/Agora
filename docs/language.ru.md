# Язык

Выберите язык отображения интерфейса Agora.

## Доступные языки

| Язык | Код | Описание |
|----------|------|-------------|
| **Системный по умолчанию** | `system` | Следовать языковым настройкам системы устройства |
| **English** | `en` | Весь текст интерфейса на английском |
| **简体中文** | `zh` | Весь текст интерфейса на упрощённом китайском |

## Как изменить

1. Перейдите в **Настройки → Язык**
2. Выберите предпочитаемый язык
3. Появится snackbar: нажмите **Перезапустить**, чтобы применить изменение немедленно

!!! note "Требуется перезапуск"
    Смена языка требует полного перезапуска приложения. Agora предложит перезапуститься после выбора нового языка — нажмите действие перезапуска в snackbar, чтобы перезагрузить приложение с новым языком.
