# 웹 검색

모델이 실시간으로 인터넷을 검색하고 웹 페이지를 가져올 수 있도록 합니다. 활성화 시, 모델은 도구 호출을 통해 자율적으로 최신 정보 조회, 사실 확인, 문서 가져오기, 주제 조사를 수행할 수 있습니다.

## 지원되는 프로바이더

| 프로바이더 | 설명 | 무료 티어 | 설정 |
|----------|-------------|-----------|-------|
| **Brave** | 개인정보 중심 검색 API | 있음 (월 2,000회 쿼리) | [api.search.brave.com](https://api.search.brave.com/) |
| **Serper** | 빠른 Google 검색 API | 있음 (월 2,500회 쿼리) | [serper.dev](https://serper.dev) |
| **Tavily** | AI 최적화 검색, LLM 에이전트용 | 있음 (월 1,000회 쿼리) | [tavily.com](https://tavily.com) |
| **SearXNG** | 자체 호스팅 메타검색 엔진 | 자체 호스팅 (무제한) | 자체 인스턴스 |

## 설정

### Brave

1. [Brave Search API](https://api.search.brave.com/)에서 API 키 발급
2. Agora에서 **설정 → 웹 검색**으로 이동
3. 검색 프로바이더로 **Brave** 선택
4. API 키 붙여넣기

### Serper

1. [serper.dev](https://serper.dev)에서 API 키 발급
2. Agora에서 **설정 → 웹 검색**으로 이동
3. **Serper** 선택
4. API 키 붙여넣기

### Tavily

1. [tavily.com](https://tavily.com)에서 API 키 발급
2. Agora에서 **설정 → 웹 검색**으로 이동
3. **Tavily** 선택
4. API 키 붙여넣기

### SearXNG

1. SearXNG 인스턴스 설정 (자체 호스팅) 또는 공개 인스턴스 사용
2. Agora에서 **설정 → 웹 검색**으로 이동
3. **SearXNG** 선택
4. 인스턴스의 **기본 URL** 입력 (예: `https://searx.be`)
5. API 키는 선택 사항 (인스턴스가 인증을 요구하는 경우에만 필요)

!!! warning "공개 인스턴스"
    공개 SearXNG 인스턴스는 종종 속도 제한이 있거나 불안정합니다. 일관된 사용을 위해 자체 호스팅을 권장합니다.

## 구성

### 최대 결과

쿼리당 가져올 검색 결과 수 설정: **1–10**. 기본값은 기기에 따라 다릅니다. 더 많은 결과는 모델에 더 많은 컨텍스트를 제공하지만 토큰 비용이 더 듭니다.

### 활성화/비활성화

웹 검색 설정 페이지에서 **웹 검색 활성화**를 토글합니다. 비활성화 시, 모델은 웹 검색 도구를 호출할 수 없습니다.

---

## 모델의 검색 사용 방식

최신 정보나 외부 정보가 필요한 질문을 하면, 모델이 자동으로 웹 검색을 호출합니다:

1. **검색**: 모델이 자체 구성한 쿼리로 검색 API 호출
2. **가져오기**: 모델이 결과 URL에서 전체 페이지 콘텐츠를 선택적으로 가져올 수 있음
3. **종합**: 모델이 결과를 읽고 응답에 통합

각 검색과 가져오기는 대화에서 인라인 도구 카드로 표시됩니다.

### 예시

```text
사용자: "Python 최신 버전이 뭔가요?"
모델: ["최신 Python 버전 2026" 검색]
       [결과 읽기]
       "Python 3.14.0이 2025년 10월에 출시되었습니다..."
```

---

## 웹 가져오기

검색 외에도, 모델은 특정 웹 페이지를 가져와서 읽을 수 있습니다. 모델이 검색 결과에서 URL을 발견하면, `web_fetch`를 호출하여 전체 페이지 콘텐츠를 가져올 수 있습니다:

- 가져온 콘텐츠는 마크다운으로 변환됨
- 모델이 이를 처리하고 관련 정보 추출
- 가져오기 결과는 도구 카드로 표시됨

---

## 개인정보 고려 사항

웹 검색 사용 시:

- 쿼리는 Agora가 아닌 검색 프로바이더(Brave, Serper 등)로 전송됨
- Agora는 검색 쿼리를 기록하거나 저장하지 않음 (대화 자체 내에서는 예외)
- SearXNG 자체 호스팅이 가장 높은 개인정보 보호 제공 — 쿼리가 자체 인프라에 머무름
