# 外觀

自訂 Agora 的外觀與風格 — 主題模式、色彩方案與風格。

## 主題模式

在淺色與深色外觀之間選擇。

| 模式 | 說明 |
|------|-------------|
| **系統** | 跟隨您裝置的系統主題（預設） |
| **淺色** | 始終使用淺色主題 |
| **深色** | 始終使用深色主題 |

變更會立即生效。

---

## 色彩方案

從一組精選的、基於 Material 3 設計原則的色彩調色盤中選擇。

| 方案 | 特色 |
|--------|-----------|
| **午夜** | 深邃藍色調，沉穩專業（預設） |
| **北歐** | 冷色調青綠與石板色，簡潔極簡 |
| **森林** | 溫暖的綠色與大地色調，自然感受 |
| **日落** | 橙色與暖色調，充滿活力 |
| **玫瑰** | 粉紅與玫瑰色點綴，柔和溫暖 |
| **薰衣草** | 紫色與紫羅蘭色調，優雅 |
| **石板** | 中性灰與藍色，低調內斂 |
| **海洋** | 深邃藍與青色，清爽 |

每個方案都有淺色和深色兩種變體。

---

## 方案風格

方案風格決定了所選調色盤中的顏色*如何*套用到 UI 元素。

| 風格 | 效果 |
|-------|--------|
| **色調焦點** | 在關鍵表面上使用主色點綴（預設） |
| **表現力** | 整體大膽使用強調色 |
| **鮮豔** | 高飽和度色彩，活潑生動的外觀 |
| **中性** | 色彩極簡，主要使用表面色調 |

---

## 動態色彩 (Android 12+)

在執行 Android 12 或更高版本的裝置上啟用後，Agora 會從您的系統桌布中提取色彩調色盤並自動套用。

這會覆蓋手動的 **色彩方案** 選擇。停用它即可使用自訂方案。

!!! info "可用性"
    動態色彩需要 Android 12 (API 31) 或更高版本。在舊版裝置上，此開關會隱藏。
