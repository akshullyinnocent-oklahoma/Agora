# توليد الصور

توليد صور من توجيهات نصية باستخدام نموذج نص إلى صورة، مباشرة داخل محادثاتك.

## ما يفعله

عند تفعيل توليد الصور، يمكن لـ Agora تحويل توجيهاتك إلى صور باستخدام نموذج نص إلى صورة مخصص (مثل DALL·E، GPT-Image، Imagen، FLUX، Stable Diffusion، Seedream، Qwen-Image، والعديد غيرها). الصورة المولدة تُعاد إلى المحادثة، بحيث يمكنك التكرار عليها تمامًا مثل أي رد آخر.

توليد الصور يستخدم **اختيار نموذج خاص به**، مستقل عن النموذج الذي تتحدث معه — لذا يمكنك الدردشة مع نموذج وتوليد الصور مع آخر.

## الإعداد

1. اذهب إلى **الإعدادات → توليد الصور**
2. بدّل **تفعيل توليد الصور** إلى وضع التشغيل
3. اضغط **نموذج** واختر نموذج نص إلى صورة
4. اختياريًا عيّن **الحجم الافتراضي** (العرض × الارتفاع)

!!! note "يعيد استخدام بيانات اعتماد المزود"
    نموذج الصورة يعيد استخدام مفتاح API و URL الأساسي لمزوده — نفس بيانات الاعتماد التي أعددتها مسبقًا تحت **الإعدادات → مزودو API**. لا يوجد مفتاح منفصل لإدخاله. يمكن أيضًا استخدام نموذج محلي (بالأوزان الصحيحة) لإبقاء التوليد على الجهاز.

## اختيار النموذج

اضغط **نموذج** لاختيار النموذج المستخدم للتوليد.

- المنتقي يعرض النماذج التي تبدو كنماذج نص إلى صورة، مصفاة من كل نماذجك المتزامنة، بحيث تبقى القائمة قصيرة.
- إذا كان النموذج الذي تريده غير مدرج (اسم غير معتاد)، فعّل **إظهار كل النماذج** للاختيار من القائمة الكاملة.
- فقط إدخال `Provider:model` متزامن بشكل صحيح يُحتسب كاختيار صالح. زامن نماذجك أولاً تحت **الإعدادات → مزودو API** / **إدارة النماذج** إذا كانت القائمة فارغة.

## الحجم الافتراضي

يعيّن أبعاد المخرجات الافتراضية، مدخلة كـ **عرض × ارتفاع** بالبكسل (مثلاً `1024` × `1024`).

- الافتراضي هو `1024 × 1024`.
- الأحجام المدعومة تعتمد على النموذج والمزود — إذا رفض نموذج حجمًا، جرب قيمة يوثقها (الخيارات الشائعة هي `1024×1024`، `1024×1792`، `1792×1024`).

## كيف يعمل

1. فعّل توليد الصور واختر نموذج صور
2. في محادثة، اطلب من المساعد إنشاء صورة
3. يوجه Agora الطلب إلى نموذج الصور المعد باستخدام بيانات اعتماد ذلك المزود
4. الصورة المولدة تعاد إلى المحادثة

!!! tip
    كن محددًا في توجيهك — صف الموضوع، الأسلوب، التكوين، والإضاءة. التوجيهات الواضحة تنتج نتائج أفضل بكثير من الغامضة.
