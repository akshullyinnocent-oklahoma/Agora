# Начало работы

Это руководство проведёт вас через установку Agora, добавление первого API-ключа и отправку первого сообщения.

## Установка

### Из F-Droid (Рекомендуется)

Agora доступна в F-Droid, магазине приложений с открытым исходным кодом для Android.

1. Установите [F-Droid](https://f-droid.org/) на ваше устройство
2. Откройте F-Droid, найдите **Agora**
3. Нажмите **Установить**

### Из релизов GitHub

1. Посетите [страницу релизов](https://github.com/newo-ether/Agora/releases)
2. Скачайте последний файл `.apk`
3. Откройте файл на устройстве и подтвердите установку при запросе

### Сборка из исходников

Если вы предпочитаете собрать самостоятельно:

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/newo-ether/Agora.git
   ```
2. Откройте проект в [Android Studio](https://developer.android.com/studio) (Ladybug или новее)
3. Синхронизируйте Gradle и выполните сборку

Требования: Android SDK 34+, JDK 17+.

## Первый запуск

При первом открытии Agora вы увидите экран приветствия с полем ввода текста. Прежде чем начать общение, необходимо настроить провайдера и API-ключ.

### Шаг 1: Добавление API-ключа

1. Нажмите значок **Настройки** (шестерёнка в правом нижнем углу) на панели навигации
2. В разделе **Сервисы** нажмите **Провайдер**
3. Выберите провайдера из списка (например, **OpenAI**, **Anthropic**, **Google**)
4. Нажмите **Добавить новый ключ**
5. Введите имя для вашего ключа (например, «Личный») и вставьте ваш API-ключ
6. Нажмите **Добавить**

??? tip "Где получить API-ключ?"
    - **Google Gemini**: [Google AI Studio](https://aistudio.google.com/apikey) — доступен бесплатный тариф
    - **OpenAI**: [Platform API Keys](https://platform.openai.com/api-keys)
    - **Anthropic**: [Console API Keys](https://console.anthropic.com/)
    - **DeepSeek**: [Platform](https://platform.deepseek.com/)
    - **OpenRouter**: [Keys page](https://openrouter.ai/keys)

    Подробности о каждом провайдере смотрите на странице [API Провайдеры](provider.ru.md).

### Шаг 2: Синхронизация моделей

1. Вернитесь в Настройки и нажмите **Модели** (в разделе **Сервисы**)
2. Нажмите **Синхронизировать от всех провайдеров**
3. Agora загружает последний список моделей для всех настроенных провайдеров
4. После синхронизации нажмите на модель, чтобы установить её как **Модель по умолчанию**

### Шаг 3: Отправка первого сообщения

1. Нажмите **стрелку назад**, чтобы вернуться на экран чата
2. Введите сообщение в поле ввода внизу
3. Нажмите **Отправить** (значок бумажного самолётика)

Модель будет передавать ответ в реальном времени.

## Макет приложения

Agora имеет чистый макет, сосредоточенный вокруг экрана чата:

### Верхняя панель

- **Заголовок диалога** — отображает название текущего диалога (нажмите для переименования)
- **Меню-гамбургер** (:material-menu:) — открывает панель диалогов
- **Меню переполнения** (:material-dots-vertical:) — настройки конкретного диалога (модель, системный промпт, параметры генерации)

### Панель диалогов

Нажмите **меню-гамбургер** или проведите вправо от левого края, чтобы открыть:

- **Строка поиска** — поиск прошлых диалогов по ключевым словам или семантический поиск
- **Список диалогов** — все диалоги, сначала новые
- **Настройки** (:material-cog:) — настройка провайдеров, моделей, промптов и прочего
- **Новый чат** — начать новый диалог

### Экран чата

- **Область сообщений** — прокручиваемая история диалога с рендерингом markdown
- **Нижняя панель** — ввод текста, выбор модели, кнопка вложения (+) и кнопка отправки
