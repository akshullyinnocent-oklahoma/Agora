## Об Agora

Agora — это BYOK Android-клиент для продвинутых пользователей ИИ:

- **Без посредников**: Прямые API-подключения, без телеметрии, без отслеживания
- **Хранение на устройстве**: Всё хранится локально в базе данных Room
- **Нелинейные диалоги**: Редактируйте любое прошлое сообщение и исследуйте альтернативные ветви
- **Агентность по умолчанию**: Многораундовый вызов инструментов с веб-поиском, выполнением команд оболочки, файловыми операциями и памятью
- **Удалённое управление**: Управляйте серверами через зашифрованный протокол Conch
- **Открытый исходный код**: Лицензия MIT, [исходный код на GitHub](https://github.com/newo-ether/Agora)
