# はじめに

このガイドでは、Agora のインストール、最初の API キーの追加、そして最初のメッセージの送信方法について説明します。

## インストール

### F-Droid から（推奨）

Agora は、オープンソースの Android アプリストアである F-Droid で入手可能です。

1. デバイスに [F-Droid](https://f-droid.org/) をインストールします
2. F-Droid を開き、**Agora** を検索します
3. **インストール** をタップします

### GitHub リリースから

1. [リリースページ](https://github.com/newo-ether/Agora/releases) にアクセスします
2. 最新の `.apk` ファイルをダウンロードします
3. デバイスでファイルを開き、表示されるプロンプトでインストールを確認します

### ソースからビルド

自分でビルドしたい場合：

1. リポジトリをクローンします：
   ```
   git clone https://github.com/newo-ether/Agora.git
   ```
2. [Android Studio](https://developer.android.com/studio)（Ladybug 以降）でプロジェクトを開きます
3. Gradle を同期してビルドします

要件: Android SDK 34+、JDK 17+。

## 初回起動

Agora を初めて開くと、テキスト入力のあるウェルカム画面が表示されます。チャットを開始する前に、プロバイダーと API キーを設定する必要があります。

### ステップ 1: API キーの追加

1. ナビゲーションバーの **設定** アイコン（右下の歯車）をタップします
2. **サービス** の下の **プロバイダー** をタップします
3. リストからプロバイダーを選択します（例：**OpenAI**、**Anthropic**、**Google**）
4. **新しいキーを追加** をタップします
5. キーの名前（例：「個人用」）を入力し、API キーを貼り付けます
6. **追加** をタップします

??? tip "API キーはどこで入手できますか？"
    - **Google Gemini**: [Google AI Studio](https://aistudio.google.com/apikey) — 無料枠あり
    - **OpenAI**: [Platform API Keys](https://platform.openai.com/api-keys)
    - **Anthropic**: [Console API Keys](https://console.anthropic.com/)
    - **DeepSeek**: [プラットフォーム](https://platform.deepseek.com/)
    - **OpenRouter**: [Keys ページ](https://openrouter.ai/keys)

    各プロバイダーの詳細については、[API プロバイダー](provider.md) ページを参照してください。

### ステップ 2: モデルの同期

1. 設定に戻り、**モデル**（**サービス** の下）をタップします
2. **すべてのプロバイダーから同期** をタップします
3. Agora は設定されたすべてのプロバイダーの最新モデルリストを取得します
4. 同期後、モデルをタップして **デフォルトモデル** として設定します

### ステップ 3: 最初のメッセージを送信

1. **戻る矢印** をタップしてチャット画面に戻ります
2. 下部の入力フィールドにメッセージを入力します
3. **送信**（紙飛行機アイコン）をタップします

モデルはリアルタイムで応答をストリーミングします。

## アプリのレイアウト

Agora はチャット画面を中心としたクリーンなレイアウトです：

### トップバー

- **会話タイトル** — 現在の会話名を表示します（タップして名前変更）
- **ハンバーガーメニュー** (:material-menu:) — 会話ドロワーを開きます
- **オーバーフローメニュー** (:material-dots-vertical:) — 会話ごとの設定（モデル、システムプロンプト、生成パラメータ）

### 会話ドロワー

**ハンバーガーメニュー** をタップするか、左端から右にスワイプして開きます：

- **検索バー** — キーワードまたはセマンティック検索で過去の会話を検索
- **会話リスト** — すべての会話、新しい順
- **設定** (:material-cog:) — プロバイダー、モデル、プロンプトなどの設定
- **新しいチャット** — 新しい会話を開始

### チャット画面

- **メッセージエリア** — マークダウンレンダリング付きのスクロール可能な会話履歴
- **下部バー** — テキスト入力、モデルセレクター、添付ボタン（+）、送信ボタン
