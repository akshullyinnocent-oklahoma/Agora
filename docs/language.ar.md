# اللغة

اختر لغة عرض واجهة Agora.

## اللغات المتاحة

| اللغة | الرمز | الوصف |
|----------|------|-------------|
| **افتراضي النظام** | `system` | اتباع إعداد لغة نظام جهازك |
| **English** | `en` | كل نص الواجهة بالإنجليزية |
| **简体中文** | `zh` | كل نص الواجهة بالصينية المبسطة |

## كيفية التغيير

1. اذهب إلى **الإعدادات → اللغة**
2. اختر لغتك المفضلة
3. يظهر شريط إشعار: اضغط **إعادة التشغيل** لتطبيق التغيير فورًا

!!! note "إعادة التشغيل مطلوبة"
    تغييرات اللغة تتطلب إعادة تشغيل كاملة للتطبيق. يطالبك Agora بإعادة التشغيل بعد اختيار لغة جديدة — اضغط إجراء إعادة التشغيل في شريط الإشعار لإعادة تحميل التطبيق باللغة الجديدة.
