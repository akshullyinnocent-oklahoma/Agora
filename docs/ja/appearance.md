# 外観

Agora のルックアンドフィール（テーマモード、カラースキーム、スタイル）をカスタマイズします。

## テーマモード

ライトまたはダークの外観を選択します。

| モード | 説明 |
|------|-------------|
| **システム** | デバイスのシステムテーマに従います（デフォルト） |
| **ライト** | 常にライトテーマを使用 |
| **ダーク** | 常にダークテーマを使用 |

変更はすぐに反映されます。

---

## カラースキーム

Material 3 デザイン原則に基づいて厳選されたカラーパレットセットから選択します。

| スキーム | 特徴 |
|--------|-----------|
| **Midnight** | 深いブルーのトーン、落ち着いたプロフェッショナルな印象（デフォルト） |
| **Nordic** | クールなティールとスレート、クリーンでミニマル |
| **Forest** | 暖かいグリーンとアースカラー、自然な雰囲気 |
| **Sunset** | オレンジと暖色系、エネルギッシュ |
| **Rose** | ピンクとローズのアクセント、柔らかく温かい |
| **Lavender** | パープルとバイオレットの色合い、エレガント |
| **Slate** | ニュートラルなグレーとブルー、控えめ |
| **Ocean** | 深いブルーとシアン、リフレッシュ |

各スキームにはライトとダークの両方のバリアントがあります。

---

## スキームスタイル

スキームスタイルは、選択したパレットの色が UI 要素に *どのように* 適用されるかを決定します。

| スタイル | 効果 |
|-------|--------|
| **Tonal Spot** | キーとなる面にプライマリカラーのアクセント（デフォルト） |
| **Expressive** | 全体にアクセントカラーを大胆に使用 |
| **Vibrant** | 高彩度の色、生き生きとした外観 |
| **Neutral** | 最小限の色、主にサーフェストーン |

---

---

## ブラー効果

**ブラー効果を有効にする** を切り替えて、UI 全体の背景ブラーとグラデーションブラーを制御します。無効にすると、ブラー効果は単色に置き換えられ、低スペックデバイスでのパフォーマンスが向上します。

!!! tip "パフォーマンス"
    古いデバイスでスクロールやアニメーション中に UI の遅延を感じる場合は、ブラー効果を無効にしてください。

---

## ダイナミックカラー（Android 12+）

Android 12 以上を搭載したデバイスで有効にすると、Agora はシステムの壁紙からカラーパレットを抽出して自動的に適用します。

これは手動の **カラースキーム** 選択を上書きします。カスタムスキームを使用するには無効にしてください。

!!! info "利用条件"
    ダイナミックカラーは Android 12（API 31）以上が必要です。古いデバイスではトグルは非表示になります。
