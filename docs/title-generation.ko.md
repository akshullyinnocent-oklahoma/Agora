# 제목 생성

첫 번째 교환을 기반으로 대화 제목을 자동으로 생성합니다.

## 기능

새 대화를 시작할 때, Agora는 첫 메시지와 모델의 응답을 기반으로 짧고 의미 있는 제목을 자동으로 생성할 수 있습니다. 이는 일반적인 "새 채팅" 제목을 대체합니다.

## 설정

1. **설정 → 제목 생성**으로 이동
2. **제목 자동 생성** 토글
3. 선택적으로 제목 생성용 **모델** 선택 (기본적으로 현재 대화 모델 사용)

!!! tip "모델 선택"
    제목 생성은 매우 적은 토큰을 사용합니다. 대화 품질에 영향을 주지 않고 저렴하고 빠른 모델(예: GPT-4o Mini 또는 로컬 모델)을 사용할 수 있습니다.

## 작동 방식

1. 새 대화에서 첫 메시지 전송
2. 모델이 응답 (평소대로)
3. 응답 완료 후, Agora가 제목 생성을 위한 별도의 작은 요청 전송
4. 생성된 제목이 저장되고 대화 목록에 표시됨

제목 생성은 대화당 한 번만, 첫 번째 교환 시에만 실행됩니다.

## 제목 생성 모델

제목 생성 전용으로 다른 모델을 사용할 수 있습니다:

- **기본값** (선택 안 함) — 대화와 동일한 모델 사용
- **특정 모델** — 대화에 사용된 모델과 관계없이 모든 제목 생성에 항상 해당 모델 사용

제목 전용 빠른 모델을 사용하면 지연 시간과 비용을 줄일 수 있습니다.
