# Генерация заголовков

Автоматически генерируйте заголовки диалогов на основе первого обмена сообщениями.

## Что это делает

Когда вы начинаете новый диалог, Agora может автоматически сгенерировать короткий, осмысленный заголовок на основе вашего первого сообщения и ответа модели. Это заменяет общий заголовок «Новый чат».

## Настройка

1. Перейдите в **Настройки → Генерация заголовков**
2. Переключите **Авто-генерация заголовков**
3. Опционально выберите **Модель** для генерации заголовков (по умолчанию используется модель текущего диалога)

!!! tip "Выбор модели"
    Генерация заголовков использует очень мало токенов. Вы можете использовать дешёвую, быструю модель (например, GPT-4o Mini или локальную модель), не влияя на качество вашего диалога.

## Как это работает

1. Вы отправляете первое сообщение в новом диалоге
2. Модель отвечает (как обычно)
3. После завершения ответа Agora отправляет отдельный, небольшой запрос для генерации заголовка
4. Сгенерированный заголовок сохраняется и отображается в списке диалогов

Генерация заголовка выполняется только один раз за диалог, при первом обмене.

## Модель для генерации заголовков

Вы можете использовать другую модель специально для генерации заголовков:

- **По умолчанию** (без выбора) — Использует ту же модель, что и диалог
- **Конкретная модель** — Всегда использует эту модель для всей генерации заголовков, независимо от того, какая модель используется для диалога

Использование выделенной быстрой модели для заголовков может снизить задержку и стоимость.
