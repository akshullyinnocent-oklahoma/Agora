# Импорт PDF

Agora может извлекать и отправлять выбранные страницы из PDF-файлов как изображения в модели с поддержкой зрения.

## Как это работает

1. Прикрепите файл PDF в чате
2. Открывается диалог, показывающий все страницы в виде миниатюр
3. Выберите, какие страницы отправить
4. Подтвердите — Agora извлекает страницы как изображения и отправляет их модели

Модель получает страницы как визуальный ввод, что позволяет ей читать и анализировать содержимое PDF.

---

## Выбор страниц

### Выбор страниц

- Каждая миниатюра страницы имеет **флажок** в левом верхнем углу
- Нажмите флажок, чтобы переключить страницу
- Кнопки **Выбрать все** / **Снять все** для массовых операций
- Верхняя панель показывает количество выбранных (например, «Выбрано 3»)
- Максимум **50 страниц** на PDF (большие файлы обрезаются)

### Полноэкранный предпросмотр

Нажмите на любую миниатюру, чтобы открыть полноэкранный просмотрщик:

| Жест | Действие |
|---------|--------|
| Свайп влево/вправо | Навигация между страницами |
| Нажатие | Показать/скрыть элементы управления |
| Щипок | Увеличить/уменьшить |
| Капсула внизу слева | Переключить выбор страницы |

Предпросмотр позволяет проверить страницы перед тем, как решить, какие отправить.

---

## Отправка страниц

После выбора страниц нажмите кнопку подтверждения. Agora:

1. Рендерит каждую выбранную страницу PDF как изображение высокого разрешения
2. Прикрепляет изображения к вашему сообщению
3. Отправляет их модели (требуется модель с поддержкой зрения)

---

## Ограничения

- **Максимум 50 страниц** на PDF — большие файлы обрезаются
- **Только изображения**: Текст не извлекается; модель читает страницы визуально
- **Требуется vision-модель**: Активная чат-модель должна поддерживать ввод изображений
- **Размер файла**: Хотя жёсткого ограничения на размер PDF нет, очень большие файлы могут рендериться медленно

---

## Варианты использования

- **Анализ документов** — загрузите контракт, отчёт или статью для просмотра моделью
- **Исследования** — делитесь конкретными страницами из научных работ
- **Перевод** — отправляйте документы на иностранном языке для перевода
- **Суммаризация** — получайте резюме длинных документов, страница за страницей
