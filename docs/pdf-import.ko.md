# PDF 가져오기

Agora는 PDF 파일에서 선택한 페이지를 이미지로 추출하여 비전 지원 모델로 전송할 수 있습니다.

## 작동 방식

1. 채팅에서 PDF 파일 첨부
2. 모든 페이지를 썸네일로 표시하는 대화 상자 열림
3. 보낼 페이지 선택
4. 확인 — Agora가 페이지를 이미지로 추출하여 모델에 전송

모델은 페이지를 비전 입력으로 받아 PDF 콘텐츠를 읽고 분석할 수 있습니다.

---

## 페이지 선택

### 페이지 선택하기

- 각 페이지 썸네일은 왼쪽 상단에 **체크박스**가 있음
- 체크박스를 탭하여 페이지 선택/해제
- **모두 선택** / **모두 해제** 버튼으로 일괄 작업
- 상단 바에 선택된 수 표시 (예: "3개 선택됨")
- PDF당 최대 **50페이지** (더 큰 파일은 잘림)

### 전체 화면 미리보기

썸네일을 탭하여 전체 화면 뷰어 열기:

| 제스처 | 동작 |
|---------|--------|
| 왼쪽/오른쪽 스와이프 | 페이지 간 이동 |
| 탭 | 오버레이 컨트롤 표시/숨기기 |
| 핀치 | 확대/축소 |
| 왼쪽 하단 캡슐 | 페이지 선택 토글 |

미리보기를 통해 전송할 페이지를 결정하기 전에 검토할 수 있습니다.

---

## 페이지 전송

페이지 선택 후 확인 버튼 탭. Agora가:

1. 선택된 각 PDF 페이지를 고해상도 이미지로 렌더링
2. 이미지를 메시지에 첨부
3. 모델로 전송 (비전 지원 모델 필요)

---

## 제한 사항

- **PDF당 최대 50페이지** — 더 큰 파일은 잘림
- **이미지 전용**: 텍스트가 추출되지 않음; 모델이 페이지를 시각적으로 읽음
- **비전 모델 필요**: 활성 채팅 모델이 이미지 입력을 지원해야 함
- **파일 크기**: 엄격한 PDF 크기 제한은 없지만, 매우 큰 파일은 렌더링이 느릴 수 있음

---

## 사용 사례

- **문서 분석** — 계약서, 보고서, 논문을 업로드하여 모델이 검토
- **연구** — 학술 논문의 특정 페이지 공유
- **번역** — 외국어 문서를 번역용으로 전송
- **요약** — 긴 문서의 페이지별 요약 받기
