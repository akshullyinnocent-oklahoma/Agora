# البحث في المحادثات

يمكن لـ Agora البحث في سجل محادثاتك بالكامل — إما بمطابقة الكلمات المفتاحية أو الاسترجاع الدلالي (المعنى) باستخدام نماذج التضمين.

## طرق البحث

### البحث بالكلمات المفتاحية

مطابقة نصية دقيقة وسريعة. يبحث البحث عن تواجد حرفي لاستعلامك في محتوى الرسائل.

**الأنسب لـ:**
- إيجاد عبارة أو مصطلح محدد
- عمليات بحث سريعة عندما تتذكر الصياغة الدقيقة
- بدون إعداد — يعمل فورًا

**القيود:**
- يفوت المرادفات والمفاهيم المرتبطة
- لا فهم للمعنى

### البحث الدلالي (RAG)

يستخدم نماذج التضمين لإيجاد الرسائل **بالمعنى**، وليس الكلمات الدقيقة. استعلام عن "كيفية إعداد قاعدة البيانات" يمكن أن يجد رسائل عن "تكوين Room" حتى لو لم تظهر كلمة "قاعدة البيانات" أبدًا.

**الأنسب لـ:**
- إيجاد المحادثات حسب الموضوع أو الفكرة
- استعلامات واسعة عندما لا تتذكر الصياغة الدقيقة
- اكتشاف مناقشات مرتبطة عبر محادثات مختلفة

**المتطلبات:**
- يجب إعداد نموذج تضمين (راجع [التضمين / RAG](embedding.md))
- يجب تخزين الرسائل مؤقتًا (توليد التضمينات)

---

## الإعداد

### 1. إضافة نموذج تضمين

راجع [التضمين / RAG](embedding.md) للإعداد المفصل. يمكنك استخدام:
- **نماذج عن بُعد** (OpenAI، Mistral، Voyage، Ollama، إلخ.)
- **نماذج محلية** (ملفات GGUF، دون اتصال كامل)

### 2. اختيار طرق البحث

في **الإعدادات → البحث في المحادثات**:

| الإعداد | الوصف |
|---------|-------------|
| **طريقة بحث النموذج** | كيف يبحث النموذج عندما يستدعي أداة `search_conversations` |
| **طريقة البحث اليدوي** | كيف يعمل شريط البحث في درج المحادثات |

عين كل منها إلى **كلمات مفتاحية** أو **دلالي (RAG)**.

### 3. تكوين نطاق البحث

| الإعداد | النطاق | الوصف |
|---------|-------|-------------|
| **رسائل السياق لكل نتيجة بحث** | 4–32 | كم رسالة محيطة تُضمن مع كل تطابق (الخطوات: 4، 8، 12، 16، 20، 24، 28، 32) |
| **الحد الأقصى لنتائج البحث** | 5–30 | العدد الأقصى للتطابقات المعادة (الخطوات: 5، 10، 15، 20، 25، 30) |
| **عتبة التشابه** | 0.0–1.0 | RAG فقط: درجة التشابه الدنيا للتطابق. أعلى = أكثر صرامة. الافتراضي: 0.5 |

### 4. تخزين الرسائل مؤقتًا

إذا كنت تستخدم RAG، اضغط **تخزين مؤقت** لتوليد تضمينات لكل الرسائل الموجودة. فعّل **التخزين المؤقت التلقائي** للحفاظ على تحديث الفهرس تلقائيًا.

## استخدام البحث

### البحث اليدوي (شريط البحث)

1. افتح **درج المحادثات** (قائمة الهامبرغر :material-menu: أو اسحب يمينًا)
2. اضغط شريط البحث في الأعلى
3. اكتب استعلامك
4. تظهر النتائج أدناه — اضغط أي نتيجة لفتح تلك المحادثة عند الرسالة المطابقة

### البحث المبادر من النموذج

عند تفعيل **الوصول إلى المحادثات السابقة** (الإعدادات → الذاكرة)، يمكن للنموذج البحث في سجلك ذاتيًا:

```text
أنت: "ماذا قررنا بشأن تصميم API الأسبوع الماضي؟"
النموذج: [يبحث "قرار تصميم API"]
       "الثلاثاء الماضي، قررنا استخدام..."
```

يظهر البحث كبطاقة أداة في المحادثة.

## عتبة التشابه

مزلاق **عتبة التشابه** (0.0 إلى 1.0) يتحكم في مدى قرب تطابق الرسالة لتُضمين في نتائج RAG:

- **منخفض (0.3–0.5)**: نتائج أكثر، قد تتضمن محتوى مرتبط بشكل فضفاض
- **متوسط (0.5–0.7)**: متوازن — افتراضي جيد
- **عالي (0.7–0.9)**: نتائج أقل، فقط تطابقات قريبة جدًا

ابدأ بالافتراضي وعدّل بناءً على نتائجك. إذا حصلت على تطابقات غير ذات صلة كثيرة، ارفع العتبة. إذا فاتتك محادثات ذات صلة، اخفضها.

---

## عرض نتائج البحث

في درج المحادثات، نتائج البحث تظهر:

- **عنوان المحادثة** (أو "بدون عنوان")
- **الرسالة المطابقة** — رسالة المستخدم أو النموذج التي تطابقت
- **وسم الدور** — مستخدم أو نموذج
- **رسائل السياق** — رسائل محيطة للسياق

اضغط نتيجة لفتح المحادثة ممررة إلى الرسالة المطابقة.
