---

## Mises à Jour

### Vérifier les Mises à Jour

Appuyez sur **Vérifier les Mises à Jour** pour interroger GitHub sur la dernière version. Agora compare votre version installée avec la dernière version et affiche le résultat :

- **Mise à jour disponible** — une version plus récente est publiée sur GitHub
- **À jour** — vous utilisez la dernière version

### Vérification Automatique des Mises à Jour

Lorsqu'elle est activée, Agora vérifie périodiquement les nouvelles versions en arrière-plan. Désactivez cette option pour arrêter les vérifications automatiques.

---

## Documentation

### Afficher les Liens de Documentation

Contrôle le **bouton d'aide flottant** (FAB) qui apparaît sur les sous-pages de paramètres. Lorsqu'il est activé, un bouton :material-help-circle: renvoie vers la page pertinente du manuel utilisateur.

Désactivez cette option si vous préférez une interface plus propre sans le FAB de documentation.

## Évaluation

Un formulaire de retour intégré vous permet d'évaluer l'application et de soumettre des commentaires. Vos retours aident à améliorer Agora.

---

## Journal de Crash

Si Agora rencontre une erreur, vous pouvez consulter le journal de crash depuis **Paramètres → À Propos → Journal de Crash**. Le journal s'affiche dans une boîte de dialogue monospace avec un bouton **Copier** pour partager les détails de diagnostic lors du signalement de bugs.
