# PDF インポート

Agora は PDF ファイルから選択したページを画像として抽出し、視覚対応モデルに送信できます。

## 仕組み

1. チャットで PDF ファイルを添付します
2. すべてのページをサムネイルとして表示するダイアログが開きます
3. 送信するページを選択します
4. 確認 — Agora はページを画像として抽出し、モデルに送信します

モデルはページを視覚入力として受け取り、PDF コンテンツを読み取って分析できます。

---

## ページ選択

### ページの選択

- 各ページサムネイルの左上隅に **チェックボックス** があります
- チェックボックスをタップしてページのオン/オフを切り替えます
- **すべて選択** / **すべて解除** ボタンで一括操作
- トップバーに選択数が表示されます（例：「3件選択中」）
- PDF あたり最大 **50ページ**（それ以上のファイルは切り詰められます）

### フルスクリーンプレビュー

任意のサムネイルをタップしてフルスクリーンビューアを開きます：

| ジェスチャー | アクション |
|---------|--------|
| 左右にスワイプ | ページ間を移動 |
| タップ | オーバーレイコントロールの表示/非表示 |
| ピンチ | ズームイン/アウト |
| 左下のカプセル | ページ選択の切り替え |

プレビューでは、送信するページを決定する前にページを検査できます。

---

## ページの送信

ページを選択した後、確認ボタンをタップします。Agora は：

1. 選択された各 PDF ページを高解像度画像としてレンダリングします
2. メッセージに画像を添付します
3. モデルに送信します（視覚対応モデルが必要です）

---

## 制限事項

- **PDF あたり最大 50ページ** — それ以上のファイルは切り詰められます
- **画像のみ**: テキストは抽出されず、モデルはページを視覚的に読み取ります
- **視覚モデルが必要**: アクティブなチャットモデルが画像入力に対応している必要があります
- **ファイルサイズ**: ハードな PDF サイズ制限はありませんが、非常に大きなファイルはレンダリングが遅くなる場合があります

---

## ユースケース

- **文書分析** — 契約書、レポート、論文をアップロードしてモデルにレビューさせる
- **研究** — 学術論文から特定のページを共有
- **翻訳** — 外国語の文書を送信して翻訳
- **要約** — 長い文書の要約をページごとに取得
