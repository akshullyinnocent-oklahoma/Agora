# 標題產生

根據第一次對話自動產生對話標題。

## 功能說明

當您開始新對話時，Agora 可以根據您的第一則訊息與模型的回應，自動產生一個簡短且有意義的標題。這會取代通用的「新對話」標題。

## 設定

1. 前往 **設定 → 標題產生**
2. 切換**自動產生標題**
3. 可選擇性地為標題產生選擇一個**模型**（預設使用目前的對話模型）

!!! tip "模型選擇"
    標題產生使用的 token 非常少。您可以使用便宜、快速的模型（如 GPT-4o Mini 或本機模型），而不會影響對話品質。

## 運作方式

1. 您在新對話中發送第一則訊息
2. 模型回應（如常）
3. 回應完成後，Agora 發送一個單獨的小型請求來產生標題
4. 產生的標題會被儲存並顯示在對話列表中

標題產生每個對話只執行一次，在第一次對話時觸發。

## 標題產生模型

您可以使用不同的模型專門用於標題產生：

- **預設**（無選擇）— 使用與對話相同的模型
- **特定模型** — 始終使用該模型進行所有標題產生，無論對話使用哪個模型

使用專用的快速模型來產生標題可以減少延遲和成本。
