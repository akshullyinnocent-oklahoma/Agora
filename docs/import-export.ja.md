# データポータビリティ

Agora はすべてのデータをデバイス上に保存し、完全なインポート/エクスポート機能を提供します。あなたがデータを所有します — データを移動し、取り出し、バックアップします。

## エクスポート

データを単一の `.agora` ファイルにエクスポートします — Agora が保存するすべてを含むポータブルアーカイブです。

### エクスポートされる内容

含めるものを選択します：

| カテゴリ | 内容 |
|----------|----------|
| **会話とメッセージ** | すべてのチャット履歴、メッセージツリー、ブランチ |
| **メモリ** | アクティブメモリとすべての保存メモリファイル |
| **システムプロンプト** | すべてのカスタムシステムプロンプトテンプレート |
| **設定** | アプリの設定と環境設定 |
| **API キー** | 設定されたすべての API キー |

!!! danger "API キーの警告"
    API キーは **平文** でエクスポートされます。`.agora` ファイルを持つ人は誰でもキーを読むことができます。宛先を信頼し、ファイルを安全に取り扱う場合にのみ API キーのエクスポートを有効にしてください。

### エクスポート方法

1. **設定 → データ管理** に移動します
2. **データをエクスポート** をタップします
3. 含めるカテゴリを選択します
4. **エクスポート** をタップします
5. `.agora` ファイルの保存先を選択します

---

## インポート

以前の `.agora` エクスポートからデータを復元します。

### インポート戦略

インポート時に、デバイスに既に存在するデータの扱い方を選択します：

| 戦略 | 動作 |
|----------|----------|
| **マージ** | 新しい項目を追加し、既存の項目を保持します。同じ ID の項目が存在する場合、インポートバージョンが上書きします。 |
| **置換** | 選択されたカテゴリの既存データをすべて消去してからインポートします。新規スタートです。 |
| **スキップ** | 競合のない項目のみをインポートします。既存の項目はそのままです。 |

!!! tip
    ほとんどの場合 **マージ** を使用します — デバイス上にあるものを保持しながら、新しいデータを安全に追加します。

### インポート方法

1. **設定 → データ管理** に移動します
2. **データをインポート** をタップします
3. `.agora` ファイルを選択します
4. インポートプレビューを確認します — ファイルの内容（エクスポート日、バージョン、コンテンツ数）を確認します
5. インポート戦略を選択します
6. **インポート** をタップします

!!! danger "API キーの警告"
    エクスポートファイルに API キーが含まれている場合、Agora はインポート前に警告します。キーは平文でインポートされます。ファイルの出所を信頼する場合にのみ続行してください。

---

## サードパーティインポート

他の AI チャットプラットフォームから会話をインポートします。

Claude と ChatGPT の両方は、データを **`.zip` アーカイブ** としてエクスポートします。Agora はその `.zip` を直接インポートします — 事前に解凍する必要はなく、Agora は生の `.json` ファイルを受け付け**ません**。

### Claude からのインポート

**1. Claude からエクスポート。** [Claude](https://claude.ai/) → **設定 → データ管理 → データをエクスポート** に移動します。Claude はアーカイブを迅速に準備します — 通常 **1時間以内** — ダウンロードリンクをメールで送信します。

!!! warning "すぐにダウンロード"
    Claude のダウンロードリンクは **すぐに期限切れになります**。メールが届いたらすぐに `.zip` を取得してください — 待ちすぎるとリンクが無効になり、新しいエクスポートをリクエストする必要があります。

**2. Agora にインポート。**

1. **設定 → データ管理 → サードパーティ → Claude からインポート** に移動します
2. エクスポートされた `.zip` ファイルを選択します
3. プレビューを確認します — 会話数とメッセージ数を確認します
4. **マージ** または **置換** 戦略を選択します
5. **インポート** をタップします

!!! note
    Agora は Claude の `.zip` エクスポートから直接会話データを読み取ります。添付ファイルは検出されプレビューに表示されますが、インポートされるのはメッセージテキストのみです — 添付ファイル自体はインポートされません。

### ChatGPT からのインポート

**1. ChatGPT からエクスポート。** [ChatGPT](https://chatgpt.com/) → **設定 → データ管理 → データをエクスポート** に移動します。ChatGPT はリクエストを処理し、準備ができたらダウンロードリンクをメールで送信します。

!!! info "お待ちください"
    ChatGPT のエクスポートは通常 **1～2日** かかります。これは正常です — 再リクエストせずにメールを待ってください。

**2. Agora にインポート。**

1. **設定 → データ管理 → サードパーティ → ChatGPT からインポート** に移動します
2. ダウンロードした `.zip` ファイルを選択します
3. プレビューを確認します
4. **マージ** または **置換** 戦略を選択します
5. **インポート** をタップします

!!! note
    ユーザーメッセージとアシスタントメッセージの両方がインポートされます。メッセージロールは保持されます。

## ファイル形式

`.agora` ファイルは JSON ベースのアーカイブです。技術に詳しい方であれば、標準ツールで検査または処理できます。形式は前方互換性と後方互換性を考慮して設計されています。

## 自動バックアップ

Agora はスケジュールに従って自動的にデータをバックアップできます。エクスポートを覚えておく必要はありません — Agora が代わりに処理します。

### 仕組み

- Android WorkManager を使用してバックグラウンドで定期的に自動バックアップが実行されます
- バックアップのタイミングになると、Agora は選択されたカテゴリを設定されたディレクトリにエクスポートします
- 通知はバックアップが失敗した場合にのみ表示されます — 成功したバックアップはサイレントです
- 古いバックアップは保持設定に基づいて自動的に削除されます

### 設定

1. **設定 → データ管理 → 自動バックアップ** に移動します
2. **自動バックアップ** をオン/オフにします
3. **バックアップ間隔** を設定します — 1日、3日、5日、1週間、1ヶ月から選択
4. **エクスポート内容** を選択します — 含めるカテゴリを選択します。API キーを含める**ことも可能**です（そのボックスをチェックすると警告が表示されます） — バックアップ場所がプライベートで安全な場合にのみ有効にしてください。API キーはデフォルトでは**含まれません**。
5. **バックアップ場所** を設定します — タップしてフォルダを選択します（デフォルトは `Download/Agora/Backup`）
6. **古いバックアップを自動削除** をオン/オフにし、**次の期間より古いものを削除** を設定します

!!! info "自動削除の制約"
    削除期間はバックアップ期間よりも長くする必要があります。例えば、毎週バックアップする場合、バックアップは1ヶ月後または1年後に自動削除できます — それより早くはできません。これにより、新しいバックアップが作成される前に唯一のバックアップが削除されるのを防ぎます。

!!! note
    自動バックアップは Android の WorkManager を使用して、アプリが閉じられたりデバイスが再起動しても信頼性を確保します。バッテリー節約のため、Doze モード中はバックアップがわずかに遅れる場合があります。

---

## ベストプラクティス

- **定期的にエクスポート** してバックアップとして保管 — ファイルは安全な場所に保管してください
- **自動バックアップを有効** にして手間のかからないスケジュール保護を
- **通常のエクスポートには API キーを含めない** — キーのエクスポートは完全なデバイス移行時にのみ有効にしてください
- **増分インポートにはマージを使用** — 置換は破壊的です
- **インポート前にプレビュー** — エクスポート日とコンテンツ数を確認して、正しいファイルであることを確認してください
