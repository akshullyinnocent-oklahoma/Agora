# استيراد PDF

يمكن لـ Agora استخراج وإرسال صفحات مختارة من ملفات PDF كصور إلى النماذج القادرة على الرؤية.

## كيف يعمل

1. أرفق ملف PDF في الدردشة
2. يفتح حوار يعرض كل الصفحات كصور مصغرة
3. اختر الصفحات المراد إرسالها
4. أكد — يستخرج Agora الصفحات كصور ويرسلها إلى النموذج

يتلقى النموذج الصفحات كمدخل رؤية، مما يتيح له قراءة وتحليل محتوى PDF.

---

## اختيار الصفحات

### اختيار الصفحات

- كل صورة مصغرة لصفحة لها **مربع اختيار** في الزاوية العليا اليسرى
- اضغط مربع الاختيار لتبديل الصفحة تحديد/إلغاء
- أزرار **تحديد الكل** / **إلغاء تحديد الكل** للعمليات الجماعية
- الشريط العلوي يعرض العدد المحدد (مثل "3 محدد")
- الحد الأقصى **50 صفحة** لكل PDF (الملفات الأكبر تقص)

### معاينة ملء الشاشة

اضغط أي صورة مصغرة لفتح عارض ملء الشاشة:

| الإيماءة | الإجراء |
|---------|--------|
| اسحب يسارًا/يمينًا | التنقل بين الصفحات |
| اضغط | إظهار/إخفاء عناصر التحكم المتراكبة |
| قرص | تكبير/تصغير |
| الكبسولة السفلية اليسرى | تبديل تحديد الصفحة |

المعاينة تتيح لك فحص الصفحات قبل تقرير أيها ترسل.

---

## إرسال الصفحات

بعد اختيار الصفحات، اضغط زر التأكيد. يقوم Agora بـ:

1. عرض كل صفحة PDF مختارة كصورة عالية الدقة
2. إرفاق الصور برسالتك
3. إرسالها إلى النموذج (يتطلب نموذجًا قادرًا على الرؤية)

---

## القيود

- **الحد الأقصى 50 صفحة** لكل PDF — الملفات الأكبر تقتص
- **صور فقط**: النص لا يستخرج؛ النموذج يقرأ الصفحات بصريًا
- **نموذج رؤية مطلوب**: نموذج الدردشة النشط يجب أن يدعم إدخالات الصور
- **حجم الملف**: بينما لا يوجد حد صارم لحجم PDF، الملفات الكبيرة جدًا قد تكون بطيئة في العرض

---

## حالات الاستخدام

- **تحليل المستندات** — ارفع عقدًا، تقريرًا، أو ورقة ليراجعها النموذج
- **البحث** — شارك صفحات محددة من أوراق أكاديمية
- **الترجمة** — أرسل مستندات بلغات أجنبية للترجمة
- **التلخيص** — احصل على ملخصات لمستندات طويلة، صفحة بصفحة
