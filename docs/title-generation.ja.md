# タイトル生成

最初のやり取りに基づいて会話タイトルを自動生成します。

## 機能

新しい会話を開始すると、Agora は最初のメッセージとモデルの応答に基づいて、短く意味のあるタイトルを自動生成できます。これにより、一般的な「新しいチャット」タイトルが置き換えられます。

## セットアップ

1. **設定 → タイトル生成** に移動します
2. **タイトルを自動生成** をオンにします
3. オプションでタイトル生成用の **モデル** を選択します（デフォルトでは現在の会話モデルを使用）

!!! tip "モデルの選択"
    タイトル生成はごくわずかなトークンしか使用しません。会話の品質に影響を与えることなく、安価で高速なモデル（GPT-4o Mini やローカルモデルなど）を使用できます。

## 仕組み

1. 新しい会話で最初のメッセージを送信します
2. モデルが（通常通り）応答します
3. 応答が完了した後、Agora はタイトルを生成するための別の小さなリクエストを送信します
4. 生成されたタイトルが保存され、会話リストに表示されます

タイトル生成は会話ごとに1回だけ、最初のやり取りで実行されます。

## タイトル生成モデル

タイトル生成専用に異なるモデルを使用できます：

- **デフォルト**（選択なし） — 会話と同じモデルを使用
- **特定のモデル** — 会話で使用されるモデルに関係なく、すべてのタイトル生成に常にそのモデルを使用

専用の高速モデルをタイトルに使用すると、レイテンシとコストを削減できます。
