# الأسئلة الشائعة

## API والمزودون

### كيف أحصل على مفتاح API؟

- **Google Gemini**: [Google AI Studio](https://aistudio.google.com/apikey) — فئة مجانية متاحة
- **OpenAI**: [Platform API Keys](https://platform.openai.com/api-keys)
- **Anthropic**: [Console API Keys](https://console.anthropic.com/)
- **DeepSeek**: [المنصة](https://platform.deepseek.com/)
- **OpenRouter**: [صفحة المفاتيح](https://openrouter.ai/keys)
- **Brave Search**: [Brave Search API](https://api.search.brave.com/)

### هل يمكنني استخدام عدة مفاتيح API لنفس المزود؟

نعم. كل مزود يدعم عدة مفاتيح مسماة. اضغط زر الاختيار لتحديد المفتاح النشط. مفيد للتبديل بين مفاتيح العمل/الشخصية أو الاحتفاظ بمفتاح احتياطي جاهز. راجع [مزودو API](provider.md#api-keys).

### كيف أضيف مزودًا مخصصًا؟

اذهب إلى الإعدادات → المزود → **+ إضافة مزود مخصص**. أدخل اسمًا و URL أساسي. أي نقطة نهاية متوافقة مع OpenAI ستعمل. راجع [المزودون المخصصون](provider.md#custom-providers).

## النماذج المحلية

### ما هي نماذج GGUF التي تعمل؟

يدعم Agora صيغة GGUF لكل من الدردشة والتضمين. يجب أن تناسب نماذج الدردشة ذاكرة الجهاز (1–8B معاملات حسب RAM). نماذج التضمين أصغر بكثير (100–500 MB). راجع [النماذج المحلية](local-model.md).

### كيف أشغل النماذج دون اتصال؟

استورد نموذج دردشة GGUF عبر الإعدادات → المزود → محلي → **استيراد نموذج GGUF**. للبحث الدلالي دون اتصال كامل، استورد أيضًا نموذج تضمين GGUF. لا حاجة لاتصال الشبكة.

### لماذا نموذجي المحلي بطيء جدًا؟

التشغيل المحلي يعمل على CPU جهازك. إنه أبطأ بطبيعته من APIs السحابية. نصائح: استخدم نماذج أصغر (1–3B معاملات)، تكميم أقل (Q4_K_M)، نوافذ سياق أقصر، وأغلق تطبيقات الخلفية.

## التضمين والبحث

### لماذا فشل اختبار نموذج التضمين لدي؟

الأسباب الشائعة:

- **اسم نموذج خاطئ** — تحقق من التهجئة الدقيقة، بما في ذلك وسوم Ollama (مثل `qwen3-embedding:8b` وليس `qwen3-embedding`)
- **URL أساسي خاطئ** — تأكد أن نقطة النهاية تدعم `/v1/embeddings`
- **مفتاح API مفقود** — بعض المزودين يتطلبون توثيقًا حتى للتضمين
- **الشبكة** — تحقق من الاتصال بنقطة النهاية

### ما الفرق بين البحث بالكلمات المفتاحية والبحث RAG؟

البحث بالكلمات المفتاحية يطابق النص الدقيق. البحث RAG (البحث الدلالي) يطابق بالمعنى — "إعداد قاعدة البيانات" يمكن أن يجد "تكوين Room" حتى بدون كلمات مشتركة. RAG يتطلب نموذج تضمين ورسائل مخزنة مؤقتًا. راجع [البحث في المحادثات](search.md).

### كيف أستخدم Ollama للتضمين؟

1. ثبّت Ollama على جهاز
2. اسحب نموذج تضمين: `ollama pull qwen3-embedding:8b`
3. في Agora، أضف نموذج تضمين عن بُعد مع الإعداد المسبق **Ollama**
4. استخدم `http://<host>:11434/v1` كـ URL أساسي
5. أدخل اسم النموذج الدقيق بما في ذلك الوسم (مثل `qwen3-embedding:8b`)
6. اترك مفتاح API فارغًا

## الذاكرة

### ما الفرق بين الذاكرة النشطة والذكريات المحفوظة؟

**الذاكرة النشطة** هي سياق واحد دائم يُضمن مع كل استدعاء API — النموذج يراه دائمًا. **الذكريات المحفوظة** هي مجموعة من الملفات المسماة التي يبحث فيها النموذج ويسترجعها عند الطلب. استخدم الذاكرة النشطة للحقائق الدائمة؛ استخدم الذكريات المحفوظة للمواد المرجعية. راجع [الذاكرة والتخزين المؤقت](memory.md).

### هل يمكن للنموذج تعديل ذكرياتي؟

نعم، إذا فعّلت **الوصول إلى الذكريات المحفوظة** و/أو **الوصول إلى الذاكرة النشطة** في الإعدادات → الذاكرة. يمكن للنموذج إنشاء وقراءة وتعديل وحذف الذكريات عبر استدعاءات الأدوات. جميع الصلاحيات معطلة افتراضيًا.

## الطرفية والأدوات

### كيف أعدّ الوصول للطرفية البعيدة؟

انشر خادم [Conch](https://github.com/newo-ether/conch) على جهازك المستهدف، ثم أضف الجهاز في الإعدادات → الطرفية مع URL ومفتاح API. راجع [الطرفية البعيدة](shell.md).

### هل اتصال الطرفية مشفر؟

نعم. Conch يستخدم تبادل مفاتيح ECDH + تشفير AES-256-GCM + توقيع HMAC-SHA256. كل حركة المرور بين Agora وخادم Conch مشفرة من الطرف إلى الطرف.

## البيانات

### كيف أنسخ بياناتي احتياطيًا؟

اذهب إلى الإعدادات → التحكم في البيانات → **تصدير البيانات**. اختر الفئات وصدر إلى ملف `.agora`. خزنه في مكان آمن. راجع [نقل البيانات](import-export.md).

### هل يمكنني الاستيراد من ChatGPT أو Claude؟

نعم. صدّر بياناتك من ChatGPT أو Claude (يوفران ملفات `.zip`)، ثم استورد في الإعدادات → التحكم في البيانات → **طرف ثالث**. استراتيجيتا الدمج والاستبدال مدعومتان. راجع [نقل البيانات](import-export.md#third-party-import).

### هل مفاتيح API مشمولة في التصديرات؟

يمكن أن تكون، لكنها اختيارية. شاشة التصدير تتيح لك تفعيل تضمين مفتاح API. يظهر تحذير عند تفعيله. المفاتيح مخزنة بنص صريح داخل ملف `.agora`، لذا ضمنها فقط لنقل الجهاز الكامل إلى وجهات موثوقة.

## عام

### أين تُخزن بياناتي؟

كل شيء مخزن محليًا على جهاز Android الخاص بك في قاعدة بيانات Room. Agora ليس لديه خوادم، لا مزامنة سحابية، لا تتبع. الرسائل تُرسل مباشرة من جهازك إلى مزود الذكاء الاصطناعي الذي تضبطه.

### هل يدعم Agora لغات متعددة؟

نعم. واجهة التطبيق تدعم **English** و **中文 (الصينية)**. الإعدادات → اللغة. إعادة التشغيل مطلوبة بعد التبديل.
