# 網頁搜尋

讓模型即時搜尋網際網路並擷取網頁。啟用後，模型可以查詢最新資訊、驗證事實、擷取文件或研究主題 — 全部透過工具呼叫自主完成。

## 支援的提供者

| 提供者 | 描述 | 免費方案 | 設定 |
|----------|-------------|-----------|-------|
| **DuckDuckGo Lite** | 匿名、無需 API 金鑰 | 是（無限制，盡力而為） | 無需設定 — 即刻運作 |
| **Brave** | 注重隱私的搜尋 API | 是（每月 2,000 次查詢） | [api.search.brave.com](https://api.search.brave.com/) |
| **Serper** | 快速的 Google 搜尋 API | 是（每月 2,500 次查詢） | [serper.dev](https://serper.dev) |
| **Tavily** | AI 最佳化搜尋，專為 LLM 代理打造 | 是（每月 1,000 次查詢） | [tavily.com](https://tavily.com) |
| **SearXNG** | 自託管的元搜尋引擎 | 自託管（無限制） | 您自己的執行個體 |

## 設定

### DuckDuckGo Lite

DuckDuckGo Lite 是**預設**搜尋提供者 — 無需 API 金鑰，即刻運作。

1. 在 Agora 中，前往 **設定 → 網頁搜尋**
2. 選擇 **DuckDuckGo Lite** 作為搜尋提供者
3. 無需金鑰或 URL — 即刻開始搜尋

!!! note "盡力而為的服務"
    DuckDuckGo Lite 使用 `lite.duckduckgo.com` 的 HTML 擷取。DDG 可能變更版面、施加速率限制或封鎖自動化請求。它作為明確的盡力而為、免金鑰選項提供。若需要可靠性，請設定下方的 API 型提供者。

### Brave

1. 從 [Brave Search API](https://api.search.brave.com/) 取得 API 金鑰
2. 在 Agora 中，前往 **設定 → 網頁搜尋**
3. 選擇 **Brave** 作為搜尋提供者
4. 貼上您的 API 金鑰

### Serper

1. 從 [serper.dev](https://serper.dev) 取得 API 金鑰
2. 在 Agora 中，前往 **設定 → 網頁搜尋**
3. 選擇 **Serper**
4. 貼上您的 API 金鑰

### Tavily

1. 從 [tavily.com](https://tavily.com) 取得 API 金鑰
2. 在 Agora 中，前往 **設定 → 網頁搜尋**
3. 選擇 **Tavily**
4. 貼上您的 API 金鑰

### SearXNG

1. 設定一個 SearXNG 執行個體（自託管）或使用公開執行個體
2. 在 Agora 中，前往 **設定 → 網頁搜尋**
3. 選擇 **SearXNG**
4. 輸入您執行個體的 **Base URL**（例如 `https://searx.be`）
5. API 金鑰為選填（僅在您的執行個體需要驗證時才需要）

!!! warning "公開執行個體"
    公開的 SearXNG 執行個體通常有速率限制或不可靠。建議自託管以獲得一致的使用體驗。

## 設定

### 最大結果數

設定每次查詢要擷取多少搜尋結果：**1–10**。預設值取決於裝置。更多結果可為模型提供更多上下文，但會消耗更多 token。

### 啟用/停用

在網頁搜尋設定頁面切換**啟用網頁搜尋**。停用時，模型無法呼叫網頁搜尋工具。

---

## 模型如何使用搜尋

當您提出需要最新或外部資訊的問題時，模型會自動呼叫網頁搜尋：

1. **搜尋**：模型使用它制定的查詢來呼叫搜尋 API
2. **擷取**：模型可以選擇性地從結果 URL 擷取完整頁面內容
3. **綜合**：模型讀取結果並將其整合到回應中

您會看到每個搜尋和擷取作為內聯工具卡片顯示在對話中。

### 範例

```text
您："Python 的最新版本是什麼？"
模型：[搜尋 "latest Python version 2026"]
       [讀取結果]
       "Python 3.14.0 於 2025 年 10 月發布..."
```

---

## 網頁擷取

除了搜尋之外，模型還可以擷取並閱讀特定網頁。當模型在搜尋結果中遇到 URL 時，它可以呼叫 `web_fetch` 來擷取完整頁面內容：

- 擷取的內容會轉換為 markdown
- 模型處理內容並提取相關資訊
- 擷取結果以工具卡片形式顯示

---

## 隱私考量

使用網頁搜尋時：

- 您的查詢會傳送到搜尋提供者（Brave、Serper 等），而非 Agora
- Agora 不會記錄或儲存您的搜尋查詢（除了在對話本身中）
- SearXNG 自託管為您提供最高的隱私保護 — 查詢保留在您自己的基礎設施上
