# 圖片生成

在對話中直接使用文字轉圖片模型，根據文字提示生成圖片。

## 功能概述

啟用圖片生成後，Agora 可以使用專用的文字轉圖片模型（如 DALL·E、GPT-Image、Imagen、FLUX、Stable Diffusion、Seedream、Qwen-Image 等）將您的提示轉換為圖片。生成的圖片會直接回傳到對話中，您可以像處理其他回覆一樣對其進行迭代修改。

圖片生成使用**獨立的模型選擇**，與您聊天的模型無關——因此您可以用一個模型聊天，同時用另一個模型生成圖片。

## 設定

1. 前往 **設定 → 圖片生成**
2. 開啟 **啟用圖片生成**
3. 點選 **模型** 並選擇一個文字轉圖片模型
4. 可選擇性設定 **預設尺寸**（寬度 × 高度）

!!! note "BYOK — 專用憑證"
    圖片生成使用**自己專屬的 API 金鑰與基礎 URL**，獨立於您的聊天提供者。這代表您可以用不同的服務來生成圖片與聊天。前往 **設定 → 圖片生成** 來設定金鑰、基礎 URL、模型與預設尺寸。

## 模型選擇

點選 **模型** 以選擇用於生成的模型。

- 選擇器會顯示看起來像是文字轉圖片模型的模型，從您所有已同步的模型中篩選出來，因此清單會保持簡短。
- 如果您想要的模型不在清單中（例如不常見的名稱），請啟用 **顯示所有模型** 以從完整清單中選擇。
- 只有正確同步的 `Provider:model` 項目才算有效選擇。如果清單為空，請先在 **設定 → API 提供者** / **管理模型** 中同步您的模型。

## 預設尺寸

設定預設的輸出尺寸，以像素為單位輸入 **寬度 × 高度**（例如 `1024` × `1024`）。

- 預設值為 `1024 × 1024`。
- 支援的尺寸取決於模型與提供者——如果模型拒絕某個尺寸，請嘗試其檔案中記錄的值（常見選項有 `1024×1024`、`1024×1792`、`1792×1024`）。

## 運作方式

1. 啟用圖片生成並選擇一個圖片模型
2. 在對話中，要求助理建立圖片
3. Agora 會使用該提供者的憑證，將請求路由到已設定的圖片模型
4. 生成的圖片會回傳到對話中

!!! tip
    在提示中明確描述——說明主體、風格、構圖與光線。清晰的提示產生的結果遠優於模糊的提示。
