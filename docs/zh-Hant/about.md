## 更新

### 檢查更新

點選 **檢查更新** 以查詢 GitHub 上的最新版本。Agora 會將您已安裝的版本與最新版本進行比對，並顯示結果：

- **有可用更新** — GitHub 上已發布較新版本
- **已是最新** — 您正在執行最新版本

### 自動檢查更新

啟用後，Agora 會定期在背景檢查新版本。關閉此開關可停用自動檢查。

---

## 文件

### 顯示文件連結

控制設定子頁面上出現的 **浮動說明按鈕** (FAB)。啟用後，:material-help-circle: 按鈕會連結到相關的使用手冊頁面。

若您偏好更簡潔的介面而不需要文件 FAB，請關閉此開關。

## 評分

內建的回饋表單讓您可以為應用程式評分並提交意見。您的回饋有助於改善 Agora。

---

## 崩潰記錄

若 Agora 發生錯誤，您可以從 **設定 → 關於 → 崩潰記錄** 檢視崩潰記錄。記錄以等寬字型對話框顯示，並提供**複製**按鈕，以便在回報錯誤時分享診斷資訊。
