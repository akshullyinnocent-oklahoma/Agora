# 데이터 이식성

Agora는 모든 데이터를 기기에 저장하며 완전한 가져오기/내보내기 기능을 제공합니다. 데이터는 여러분의 것입니다 — 가져오고, 내보내고, 백업하세요.

## 내보내기

데이터를 단일 `.agora` 파일로 내보냅니다 — Agora가 저장하는 모든 것을 포함하는 이식 가능한 아카이브입니다.

### 내보내기 대상

포함할 항목을 선택합니다:

| 카테고리 | 내용 |
|----------|----------|
| **대화 & 메시지** | 모든 채팅 기록, 메시지 트리, 분기 |
| **메모리** | 활성 메모리 및 모든 저장된 메모리 파일 |
| **시스템 프롬프트** | 모든 사용자 지정 시스템 프롬프트 템플릿 |
| **설정** | 앱 구성 및 환경 설정 |
| **API 키** | 구성된 모든 API 키 |

!!! danger "API 키 경고"
    API 키는 **평문으로** 내보내집니다. `.agora` 파일을 가진 사람은 누구나 키를 읽을 수 있습니다. 대상을 신뢰하고 파일을 안전하게 처리할 수 있는 경우에만 API 키 내보내기를 활성화하세요.

### 내보내기 방법

1. **설정 → 데이터 제어**로 이동
2. **데이터 내보내기** 탭
3. 포함할 카테고리 선택
4. **내보내기** 탭
5. `.agora` 파일을 저장할 위치 선택

## 가져오기

이전 `.agora` 내보내기에서 데이터를 복원합니다.

### 가져오기 전략

가져올 때, 기기에 이미 존재하는 데이터를 Agora가 처리하는 방식을 선택합니다:

| 전략 | 동작 |
|----------|----------|
| **병합** | 새 항목 추가, 기존 항목 유지. 동일한 ID의 항목이 있으면 가져오기 버전이 덮어씁니다. |
| **교체** | 선택한 카테고리의 모든 기존 데이터를 지우고 가져오기. 새로 시작. |
| **건너뛰기** | 충돌이 없는 항목만 가져오기. 기존 항목은 그대로 유지. |

!!! tip
    대부분의 경우 **병합**을 사용하세요 — 기기에 이미 있는 데이터를 보존하면서 안전하게 새 데이터를 추가합니다.

### 가져오기 방법

1. **설정 → 데이터 제어**로 이동
2. **데이터 가져오기** 탭
3. `.agora` 파일 선택
4. 가져오기 미리보기 검토 — 파일 내용 확인 (내보내기 날짜, 버전, 콘텐츠 수)
5. 가져오기 전략 선택
6. **가져오기** 탭

!!! danger "API 키 경고"
    내보내기 파일에 API 키가 포함된 경우, Agora가 가져오기 전에 경고합니다. 키는 평문으로 가져옵니다. 파일 출처를 신뢰하는 경우에만 진행하세요.

## 서드파티 가져오기

다른 AI 채팅 플랫폼에서 대화를 가져옵니다.

Claude와 ChatGPT는 모두 데이터를 **`.zip` 아카이브**로 내보냅니다. Agora는 해당 `.zip`을 직접 가져옵니다 — 먼저 압축을 풀 필요가 없으며, Agora는 개별 `.json` 파일을 **허용하지 않습니다**.

### Claude에서 가져오기

**1. Claude에서 내보내기.** [Claude](https://claude.ai/) → **설정 → 데이터 제어 → 데이터 내보내기**로 이동합니다. Claude는 보통 **1시간 이내에** 아카이브를 준비하고 다운로드 링크를 이메일로 보냅니다.

!!! warning "신속히 다운로드"
    Claude의 다운로드 링크는 **빠르게 만료됩니다**. 이메일이 도착하는 즉시 `.zip`을 받으세요 — 너무 오래 기다리면 링크가 만료되어 새 내보내기를 요청해야 합니다.

**2. Agora로 가져오기.**

1. **설정 → 데이터 제어 → 서드파티 → Claude에서 가져오기**로 이동
2. 내보낸 `.zip` 파일 선택
3. 미리보기 검토 — 대화 수 및 메시지 수 확인
4. **병합** 또는 **교체** 전략 선택
5. **가져오기** 탭

!!! note
    Agora는 Claude의 `.zip` 내보내기에서 직접 대화 데이터를 읽습니다. 첨부 파일은 감지되어 미리보기에 표시되지만, 메시지 텍스트만 가져오며 첨부 파일 자체는 가져오지 않습니다.

### ChatGPT에서 가져오기

**1. ChatGPT에서 내보내기.** [ChatGPT](https://chatgpt.com/) → **설정 → 데이터 제어 → 데이터 내보내기**로 이동합니다. ChatGPT가 요청을 처리하고 준비가 되면 다운로드 링크를 이메일로 보냅니다.

!!! info "기다려 주세요"
    ChatGPT 내보내기는 일반적으로 **1–2일** 소요됩니다. 이는 정상입니다 — 재요청하지 말고 이메일을 기다리세요.

**2. Agora로 가져오기.**

1. **설정 → 데이터 제어 → 서드파티 → ChatGPT에서 가져오기**로 이동
2. 다운로드한 `.zip` 파일 선택
3. 미리보기 검토
4. **병합** 또는 **교체** 전략 선택
5. **가져오기** 탭

!!! note
    사용자 및 어시스턴트 메시지가 모두 가져와집니다. 메시지 역할이 보존됩니다.

## 파일 형식

`.agora` 파일은 JSON 기반 아카이브입니다. 기술적인 지식이 있다면 표준 도구로 검사하거나 처리할 수 있습니다. 형식은 상위 및 하위 호환성을 위해 설계되었습니다.

## 자동 백업

Agora는 일정에 따라 자동으로 데이터를 백업할 수 있습니다. 내보내기를 기억할 필요가 없습니다 — Agora가 처리합니다.

### 작동 방식

- 자동 백업은 Android WorkManager를 사용하여 백그라운드에서 주기적으로 실행됨
- 백업 시점이 되면, Agora가 선택한 카테고리를 구성된 디렉토리로 내보냄
- 백업 실패 시에만 알림 표시 — 성공적인 백업은 조용히 진행됨
- 보존 설정에 따라 오래된 백업이 자동 삭제됨

### 구성

1. **설정 → 데이터 제어 → 자동 백업**으로 이동
2. **자동 백업** 토글 켜기/끄기
3. **백업 주기** 설정 — 1일, 3일, 5일, 1주, 또는 1개월 선택
4. **내보내기 내용** 선택 — 포함할 카테고리 선택. API 키 **포함 가능** (해당 상자 선택 시 경고 표시) — 백업 위치가 비공개이고 안전한 경우에만 활성화하세요. API 키는 기본적으로 **포함되지 않습니다**.
5. **백업 위치** 설정 — 탭하여 폴더 선택 (기본값: `Download/Agora/Backup`)
6. **오래된 백업 자동 삭제** 토글 켜기/끄기, 그리고 **다음보다 오래된 항목 삭제** 기간 설정

!!! info "자동 삭제 제약 조건"
    삭제 기간은 백업 기간보다 길어야 합니다. 예를 들어 매주 백업하는 경우, 백업은 1개월 또는 1년 후 자동 삭제될 수 있습니다 — 더 짧을 수 없습니다. 이는 새 백업이 생성되기 전에 유일한 백업이 삭제되는 것을 방지합니다.

!!! note
    자동 백업은 Android의 WorkManager를 사용하여 앱이 닫히거나 기기가 다시 시작되더라도 안정성을 보장합니다. Doze 모드 중에는 배터리 절약을 위해 백업이 약간 지연될 수 있습니다.

---

## 모범 사례

- **정기적으로 내보내기**하여 백업 — 파일을 안전한 곳에 보관
- **자동 백업 활성화**하여 수동 개입 없는 예약 보호
- **일상적인 내보내기에 API 키 포함하지 않기** — 전체 기기 마이그레이션에만 키 내보내기 활성화
- **증분 가져오기에 병합 사용** — 교체는 파괴적입니다
- **가져오기 전 미리보기** — 내보내기 날짜와 콘텐츠 수를 확인하여 올바른 파일인지 확인
