# 語言

選擇 Agora 介面的顯示語言。

## 可用語言

| 語言 | 代碼 | 說明 |
|----------|------|-------------|
| **系統預設** | `system` | 跟隨您裝置的系統語言設定 |
| **英文** | `en` | 所有 UI 文字以英文顯示 |
| **简体中文** | `zh` | 所有 UI 文字以簡體中文顯示 |

## 如何變更

1. 前往 **設定 → 語言**
2. 選擇您偏好的語言
3. 會出現 Snackbar：點選 **重新啟動** 以立即套用變更

!!! note "需要重新啟動"
    語言變更需要完整重新啟動應用程式。Agora 會在您選擇新語言後提示您重新啟動 — 點選 Snackbar 中的重新啟動動作，以新語言重新載入應用程式。
