## Agora について

Agora は AI パワーユーザー向けの BYOK Android クライアントです：

- **仲介者なし**: API 直接接続、テレメトリなし、トラッキングなし
- **デバイス内ストレージ**: すべてのデータは Room データベースにローカル保存
- **非線形会話**: 過去のメッセージを編集して代替ブランチを探索
- **デフォルトでエージェント対応**: ウェブ検索、シェル実行、ファイル操作、メモリを備えたマルチラウンドツール呼び出し
- **リモート制御**: 暗号化された Conch プロトコルでサーバーを管理
