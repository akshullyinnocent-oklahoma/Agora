# البدء

يوجهك هذا الدليل خلال تثبيت Agora، إضافة أول مفتاح API، وإرسال أول رسالة.

## التثبيت

### من F-Droid (موصى به)

Agora متاح على F-Droid، متجر تطبيقات Android مفتوح المصدر.

1. ثبّت [F-Droid](https://f-droid.org/) على جهازك
2. افتح F-Droid، ابحث عن **Agora**
3. اضغط **تثبيت**

### من إصدارات GitHub

1. زر صفحة [الإصدارات](https://github.com/newo-ether/Agora/releases)
2. حمّل أحدث ملف `.apk`
3. افتح الملف على جهازك وأكد التثبيت عند الطلب

### البناء من المصدر

إذا كنت تفضل البناء بنفسك:

1. استنسخ المستودع:
   ```
   git clone https://github.com/newo-ether/Agora.git
   ```
2. افتح المشروع في [Android Studio](https://developer.android.com/studio) (Ladybug أو أحدث)
3. زامن Gradle وابني

المتطلبات: Android SDK 34+، JDK 17+.

## التشغيل الأول

عند فتح Agora لأول مرة، سترى شاشة ترحيب مع حقل إدخال نصي. قبل أن تتمكن من الدردشة، تحتاج إلى إعداد مزود ومفتاح API.

### الخطوة 1: إضافة مفتاح API

1. اضغط أيقونة **الإعدادات** (الترس في أسفل اليمين) في شريط التنقل
2. تحت **الخدمات**، اضغط **المزود**
3. اختر مزودًا من القائمة (مثل **OpenAI**، **Anthropic**، **Google**)
4. اضغط **إضافة مفتاح جديد**
5. أدخل اسمًا لمفتاحك (مثل "شخصي") والصق مفتاح API الخاص بك
6. اضغط **إضافة**

??? tip "من أين أحصل على مفتاح API؟"
    - **Google Gemini**: [Google AI Studio](https://aistudio.google.com/apikey) — فئة مجانية متاحة
    - **OpenAI**: [Platform API Keys](https://platform.openai.com/api-keys)
    - **Anthropic**: [Console API Keys](https://console.anthropic.com/)
    - **DeepSeek**: [المنصة](https://platform.deepseek.com/)
    - **OpenRouter**: [صفحة المفاتيح](https://openrouter.ai/keys)

    راجع صفحة [مزودو API](provider.md) للحصول على تفاصيل كل مزود.

### الخطوة 2: مزامنة النماذج

1. عد إلى الإعدادات واضغط **النماذج** (تحت **الخدمات**)
2. اضغط **مزامنة من جميع المزودين**
3. يجلب Agora أحدث قائمة نماذج لكل المزودين المضبوطين
4. بعد المزامنة، اضغط على نموذج لتعيينه **كنموذج افتراضي**

### الخطوة 3: إرسال أول رسالة

1. اضغط **السهم الخلفي** للعودة إلى شاشة المحادثة
2. اكتب رسالة في حقل الإدخال في الأسفل
3. اضغط **إرسال** (أيقونة الطائرة الورقية)

سيقوم النموذج ببث استجابته في الوقت الفعلي.

## تخطيط التطبيق

يتميز Agora بتصميم نظيف يتمحور حول شاشة المحادثة:

### الشريط العلوي

- **عنوان المحادثة** — يعرض اسم المحادثة الحالية (اضغط لإعادة التسمية)
- **قائمة الهامبرغر** (:material-menu:) — تفتح درج المحادثات
- **قائمة الفائض** (:material-dots-vertical:) — إعدادات لكل محادثة (النموذج، توجيه النظام، معاملات التوليد)

### درج المحادثات

اضغط **قائمة الهامبرغر** أو اسحب يمينًا من الحافة اليسرى للفتح:

- **شريط البحث** — ابحث عن المحادثات السابقة بالكلمات المفتاحية أو البحث الدلالي
- **قائمة المحادثات** — جميع المحادثات، الأحدث أولاً
- **الإعدادات** (:material-cog:) — إعداد المزودين، النماذج، التوجيهات، والمزيد
- **محادثة جديدة** — ابدأ محادثة جديدة

### شاشة المحادثة

- **منطقة الرسائل** — سجل محادثة قابل للتمرير مع عرض markdown
- **الشريط السفلي** — إدخال نصي، محدد النموذج، زر المرفقات (+)، وزر الإرسال
