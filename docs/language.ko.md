# 언어

Agora 인터페이스의 표시 언어를 선택하세요.

## 사용 가능한 언어

| 언어 | 코드 | 설명 |
|----------|------|-------------|
| **시스템 기본값** | `system` | 기기의 시스템 언어 설정을 따릅니다 |
| **English** | `en` | 모든 UI 텍스트를 영어로 표시 |
| **简体中文** | `zh` | 모든 UI 텍스트를 간체 중국어로 표시 |

## 변경 방법

1. **설정 → 언어**로 이동
2. 원하는 언어 선택
3. 스낵바가 나타납니다: **재시작**을 탭하여 즉시 변경 적용

!!! note "재시작 필요"
    언어 변경은 전체 앱 재시작이 필요합니다. 새 언어 선택 후 Agora가 재시작을 요청합니다 — 스낵바에서 재시작 작업을 탭하여 새 언어로 앱을 다시 로드하세요.
