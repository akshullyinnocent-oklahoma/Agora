# 대화 검색

Agora는 전체 대화 기록을 검색할 수 있습니다 — 키워드 매칭 또는 임베딩 모델을 사용한 시맨틱(의미 기반) 검색을 통해.

## 검색 방법

### 키워드 검색

빠르고 정확한 텍스트 매칭. 검색은 메시지 내용에서 쿼리의 문자 그대로의 출현을 찾습니다.

**적합한 경우:**
- 특정 구문이나 용어 찾기
- 정확한 문구를 기억할 때 빠른 조회
- 설정 불필요 — 즉시 작동

**제한 사항:**
- 동의어와 관련 개념을 놓침
- 의미 이해 불가

### 시맨틱 검색 (RAG)

임베딩 모델을 사용하여 정확한 단어가 아닌 **의미**로 메시지를 찾습니다. "데이터베이스 설정 방법"에 대한 쿼리가 "데이터베이스"라는 단어가 전혀 등장하지 않아도 "Room 구성"에 대한 메시지를 찾을 수 있습니다.

**적합한 경우:**
- 주제나 테마별 대화 찾기
- 정확한 표현을 기억하지 못하는 광범위한 쿼리
- 서로 다른 대화에서 관련 토론 발견

**요구 사항:**
- 임베딩 모델이 구성되어 있어야 함 ([임베딩 / RAG](embedding.ko.md) 참조)
- 메시지가 캐시되어 있어야 함 (임베딩 생성됨)

## 설정하기

### 1. 임베딩 모델 추가

자세한 설정은 [임베딩 / RAG](embedding.ko.md)를 참조하세요. 사용 가능한 옵션:
- **원격 모델** (OpenAI, Mistral, Voyage, Ollama 등)
- **로컬 모델** (GGUF 파일, 완전 오프라인)

### 2. 검색 방법 선택

**설정 → 대화 검색**에서:

| 설정 | 설명 |
|---------|-------------|
| **모델 검색 방법** | 모델이 `search_conversations` 도구를 호출할 때 검색하는 방식 |
| **수동 검색 방법** | 대화 서랍의 검색 바 작동 방식 |

각각을 **키워드** 또는 **시맨틱 (RAG)** 으로 설정하세요.

### 3. 검색 범위 구성

| 설정 | 범위 | 설명 |
|---------|-------|-------------|
| **검색 히트당 컨텍스트 메시지** | 4–32 | 각 일치 항목과 함께 포함할 주변 메시지 수 (단계: 4, 8, 12, 16, 20, 24, 28, 32) |
| **최대 검색 결과** | 5–30 | 반환할 최대 일치 항목 수 (단계: 5, 10, 15, 20, 25, 30) |
| **유사도 임계값** | 0.0–1.0 | RAG 전용: 일치 항목에 대한 최소 유사도 점수. 높을수록 더 엄격. 기본값: 0.5 |

### 4. 메시지 캐시

RAG를 사용하는 경우, **캐시**를 탭하여 모든 기존 메시지에 대한 임베딩을 생성합니다. **자동 캐시**를 활성화하여 인덱스를 자동으로 업데이트합니다.

## 검색 사용하기

### 수동 검색 (검색 바)

1. **대화 서랍** 열기 (햄버거 메뉴 :material-menu: 또는 오른쪽으로 스와이프)
2. 상단의 검색 바 탭
3. 쿼리 입력
4. 아래에 결과 표시 — 결과를 탭하여 해당 메시지에서 대화 열기

### 모델 주도 검색

**과거 대화 접근**이 활성화된 경우 (설정 → 메모리), 모델이 자율적으로 기록을 검색할 수 있습니다:

```text
사용자: "지난주에 API 디자인에 대해 무엇을 결정했나요?"
모델: ["API 디자인 결정" 검색]
       "지난 화요일에 우리는...을 사용하기로 결정했습니다"
```

검색은 대화에서 도구 카드로 나타납니다.

## 유사도 임계값

**유사도 임계값** 슬라이더 (0.0 ~ 1.0)는 메시지가 RAG 결과에 포함되기 위해 얼마나 밀접하게 일치해야 하는지 제어합니다:

- **낮음 (0.3–0.5)**: 더 많은 결과, 느슨하게 관련된 콘텐츠 포함 가능
- **중간 (0.5–0.7)**: 균형 — 좋은 기본값
- **높음 (0.7–0.9)**: 더 적은 결과, 매우 가까운 일치만

기본값으로 시작하여 결과에 따라 조정하세요. 관련 없는 일치가 너무 많으면 임계값을 높이고, 관련 대화를 놓치면 낮추세요.

---

## 검색 결과 표시

대화 서랍에서 검색 결과는 다음을 표시합니다:

- **대화 제목** (또는 "제목 없음")
- **일치하는 메시지** — 일치한 사용자 또는 모델 메시지
- **역할 레이블** — 사용자 또는 모델
- **컨텍스트 메시지** — 컨텍스트를 위한 주변 메시지

결과를 탭하여 일치하는 메시지로 스크롤된 대화를 엽니다.
