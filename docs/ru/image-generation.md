# Генерация изображений

Генерируйте изображения из текстовых промптов с помощью модели text-to-image прямо в ваших диалогах.

## Что это делает

Когда генерация изображений включена, Agora может превращать ваши промпты в изображения с помощью выделенной модели text-to-image (такой как DALL·E, GPT-Image, Imagen, FLUX, Stable Diffusion, Seedream, Qwen-Image и многих других). Сгенерированное изображение возвращается в диалог, так что вы можете итерировать над ним так же, как и над любым другим ответом.

Генерация изображений использует **собственный выбор модели**, независимый от модели, с которой вы общаетесь — так что вы можете общаться с одной моделью и генерировать изображения с помощью другой.

## Настройка

1. Перейдите в **Настройки → Генерация изображений**
2. Включите переключатель **Включить генерацию изображений**
3. Нажмите **Модель** и выберите модель text-to-image
4. Опционально установите **Размер по умолчанию** (ширина × высота)

!!! note "BYOK — выделенные учётные данные"
    Генерация изображений использует **собственный выделенный API-ключ и базовый URL**, независимые от ваших чат-провайдеров. Это значит, вы можете использовать другой сервис для изображений, чем для чата. Перейдите в **Настройки → Генерация изображений**, чтобы настроить ключ, базовый URL, модель и размер по умолчанию.

## Выбор модели

Нажмите **Модель**, чтобы выбрать модель, используемую для генерации.

- Выборщик показывает модели, похожие на text-to-image, отфильтрованные из всех ваших синхронизированных моделей, так что список остаётся коротким.
- Если нужной модели нет в списке (необычное имя), включите **Показать все модели**, чтобы выбрать из полного списка.
- Только правильно синхронизированная запись `Провайдер:модель` считается действительным выбором. Синхронизируйте модели сначала в **Настройки → API Провайдеры** / **Управление моделями**, если список пуст.

## Размер по умолчанию

Устанавливает размеры вывода по умолчанию, вводимые как **ширина × высота** в пикселях (например, `1024` × `1024`).

- По умолчанию `1024 × 1024`.
- Поддерживаемые размеры зависят от модели и провайдера — если модель отклоняет размер, попробуйте значение из её документации (распространённые варианты: `1024×1024`, `1024×1792`, `1792×1024`).

## Как это работает

1. Включите генерацию изображений и выберите модель изображений
2. В диалоге попросите ассистента создать изображение
3. Agora направляет запрос настроенной модели изображений, используя учётные данные этого провайдера
4. Сгенерированное изображение возвращается в диалог

!!! tip
    Будьте конкретны в промпте — опишите объект, стиль, композицию и освещение. Чёткие промпты дают значительно лучшие результаты, чем расплывчатые.
