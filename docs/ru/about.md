## Обновления

### Проверить обновления

Нажмите **Проверить обновления**, чтобы запросить GitHub о последнем релизе. Agora сравнивает вашу установленную версию с последним релизом и показывает результат:

- **Доступно обновление** — на GitHub опубликована более новая версия
- **Актуальная версия** — вы используете последний релиз

### Автоматическая проверка обновлений

При включении Agora периодически проверяет наличие новых релизов в фоновом режиме. Отключите этот переключатель, чтобы отключить автоматические проверки.

---

## Документация

### Показывать ссылки на документацию

Управляет **плавающей кнопкой помощи** (FAB), которая появляется на подстраницах настроек. При включении кнопка :material-help-circle: ссылается на соответствующую страницу руководства пользователя.

Отключите этот переключатель, если предпочитаете более чистый интерфейс без FAB документации.

## Журнал сбоев

Если в Agora произошла ошибка, вы можете просмотреть журнал сбоев в **Настройки → О приложении → Журнал сбоев**. Журнал отображается в моноширинном диалоге с кнопкой **Копировать** для передачи диагностической информации при сообщении об ошибках.
